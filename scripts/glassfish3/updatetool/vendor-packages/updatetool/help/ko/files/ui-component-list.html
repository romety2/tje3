구성 요소 목록

구성 요소 목록은 도구 모음 아래의 업데이트 도구 창 오른쪽에 표시됩니다. 사용 가능한 애드온, 사용 가능한 업데이트 또는 설치 이미지의 설치된 구성 요소 중 사용자가 무엇을 보고 있는가에 따라 다른 정보가 표시됩니다.

  • 사용 가능한 애드온 보기

  • 사용 가능한 업데이트 보기

  • 설치된 구성 요소 보기

또한 구성 요소 목록을 마우스 오른쪽 버튼으로 누르면(Macintosh의 경우 Control 키를 누른 채 누르면) 전체 구성 요소 목록에 작동하는 명령과 구성 요소 목록에서 선택한 항목을 제공하는 컨텍스트 메뉴가 표시됩니다.

다운로드

업데이트의 설치 크기(근사치). 수정된 파일만 업데이트 내에 제공되므로 업데이트의 크기는 일반적으로 전체 구성 요소의 설치 크기보다 작습니다.

수정될 파일이 없는 경우 변경 값은 0이지만, 파일 허가, 구성 요소 설명 및 기타 메타데이터와 같은 속성은 변경됩니다.

업데이트의 결과 파일이 제거되거나 하나 이상의 파일의 크기가 작아진 경우에 변경 값은 음수입니다.
