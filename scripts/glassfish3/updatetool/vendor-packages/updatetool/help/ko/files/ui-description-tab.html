설명 탭

설명 탭은 구성 요소 목록 아래의 업데이트 도구 창 오른쪽에 표시됩니다. 특정 구성 요소를 선택하면 설명 탭에 구성 요소에 대한 세부 정보가 표시됩니다.

선택된 구성 요소의 세부 정보:

제목

설명

설명

구성 요소의 개요.

패키지 이름

구성 요소의 기본 패키지 이름.

버전

구성 요소의 버전.

설치 크기

구성 요소의 설치에 사용되는 디스크 공간의 양.

게시 날짜

이 버전의 구성 요소가 저장소에 게시된 날짜 및 시간(설치 이미지에 설치된 날짜와 시간이 아님).

소스

패키지를 가져왔거나(설치된 구성 요소의 경우) 가져올(애드온 및 업데이트의 경우) 저장소의 이름.

고유 식별자

저장소에 존재하는 이 구성 요소에 대한 정규화된 식별자.
