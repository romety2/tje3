주요 소프트웨어 영역

기존 응용 프로그램 이미지와 연결된 주요 소프트웨어가 있는 경우 응용 프로그램 이미지 영역 아래에서 업데이트 도구 창의 왼쪽에 주요 소프트웨어 영역이 표시됩니다. 다음과 같은 주요 소프트웨어의 두 개 목록이 표시됩니다.

  • 응용 프로그램: 이미 설치된 응용 프로그램을 보완하는 응용 프로그램.

  • 애드온: 이미 설치된 응용 프로그램에 특히 중요한 애드온.

응용 프로그램을 선택하면 도구 모음 아래에서 업데이트 도구 창의 오른쪽에 새 응용 프로그램 목록이 표시됩니다. 목록에는 주요 응용 프로그램의 이름, 버전 및 다운로드 크기가 표시됩니다. 새 응용 프로그램 목록 아래에 현재 선택된 새 응용 프로그램을 설명하는 영역이 있고, 응용 프로그램에 대한 자세한 내용을 설명하는 자세한 내용 버튼과 응용 프로그램을 설치할 수 있는 설치 버튼이 제공됩니다.

애드온을 선택하면 도구 모음 바로 아래에서 업데이트 도구 창의 오른쪽에 새 애드온 목록이 표시됩니다. 목록에는 주요 애드온의 부모 응용 프로그램, 이름, 버전 및 다운로드 크기가 표시됩니다. 새 애드온 목록 아래에 현재 선택된 새 애드온을 설명하는 영역이 있고, 애드온에 대한 자세한 내용을 설명하는 자세한 내용 버튼과 애드온을 설치할 수 있는 설치 버튼이 제공됩니다.
