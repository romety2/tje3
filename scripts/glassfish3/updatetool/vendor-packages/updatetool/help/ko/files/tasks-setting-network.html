네트워크 프록시 기본 설정을 지정하는 방법

업데이트 도구 및 데스크탑 알림 프로그램을 인터넷에 연결하는 방법에 영향을 미치는 기본 설정을 지정할 수 있습니다.

업데이트 도구를 통해 네트워크 프록시 기본 설정 지정

  1. 도우 모음에서 기본 설정을 누릅니다.

    업데이트 도구는 기본 설정 창을 표시합니다.

  2. 인터넷에 직접 연결하는 경우 "인터넷에 직접 연결"을 누릅니다.

    프록시 서버를 통해 인터넷에 연결하는 경우 "수동 프록시 구성"을 누릅니다. 그런 다음 연결하는 데 사용할 프록시 서버에 대한 다음 정보를 제공합니다.

    • HTTP 프록시: 프록시 서버의 전체 이름을 입력합니다.
    • 포트: 프록시 서버에서 연결하는 데 사용하는 포트 번호를 입력합니다.
    • 프록시 인증 필요: 프록시 서버에서 로그인을 요구하는 경우 이 옵션을 선택하고 사용자 이름 및 비밀번호 필드를 입력합니다. 그렇지 않은 경우 이 옵션을 선택하지 않습니다.
    • 사용자 이름: 프록시 서버에 로그인하는 데 사용하는 사용자 이름을 입력합니다.
    • 비밀번호: 프록시 서버에 로그인하는 데 사용하는 비밀번호를 입력합니다.

  3. 확인을 누릅니다.

알림 프로그램 메뉴를 통해 네트워크 프록시 기본 설정 지정

이 방법은 Mac OS X에서 지원되지 않습니다.

  1. 데스크탑 알림 영역에서 알림 프로그램 아이콘을 마우스 오른쪽 버튼으로 누른 다음 기본 설정을 누릅니다.

    기본 설정 창이 표시됩니다.

  2. 위와 같이 계속합니다.
