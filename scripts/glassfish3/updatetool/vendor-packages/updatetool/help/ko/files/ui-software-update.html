소프트웨어 업데이트 인터페이스

소프트웨어 업데이트 창에는 다음 구성 요소가 포함됩니다.

구성 요소

설명

소프트웨어 업데이트 표

업데이트를 사용할 수 있는 설치 이미지가 나열됩니다. 크기 열에는 각 이미지의 업데이트 다운로드 크기가 표시됩니다.

새로 고침 버튼

모든 설치 이미지의 원격 저장소에서 업데이트를 확인합니다.

새 소프트웨어 및 애드온 표

현재 설치 이미지의 소프트웨어와 관련되고 새로 사용 가능한 소프트웨어 및 애드온이 나열됩니다.

정보 상자

소프트웨어 업데이트 표 또는 새 소프트웨어 및 애드온 표에서 선택한 항목에 대한 추가 정보를 제공합니다.

사용권 조항 확인란

설치하기 위해 선택한 업데이트, 소프트웨어 또는 애드온에서 요구하는 사용권 조항을 수락하려면 이 확인란을 누릅니다. 사용권 조항을 확인하려면 "사용권 조항" 링크를 누릅니다.

세부 정보 관리 버튼

패키지별로 업데이트를 관리하기 위해 사용할 수 있는 업데이트 도구 인터페이스를 시작합니다.

닫기 버튼

업데이트나 설치를 수행하지 않고 소프트웨어 업데이트 도구를 종료합니다.

설치 버튼

선택한 업데이트, 소프트웨어 또는 애드온을 설치합니다.
