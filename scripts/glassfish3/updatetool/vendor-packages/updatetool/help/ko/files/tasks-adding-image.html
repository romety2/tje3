기존 이미지를 응용 프로그램 이미지 목록에 추가하는 방법

대부분의 경우, 소프트웨어를 다운로드 및 설치하면 설치된 이미지가 업데이트 도구에 추가됩니다. 하지만, 다른 시스템에서 설치된 이미지를 복사하여 사용자 시스템에서 사용하는 경우가 있습니다. 설치된 이미지를 사용자 시스템으로 복사한 후에는 설치된 이미지 목록에 추가해야만 업데이트 도구가 해당 이미지를 최신 이미지로 유지할 수 있습니다.

  1. 도구 모음에서 열기를 누릅니다.

    업데이트 도구가 설치 이미지 디렉토리 선택 창을 표시합니다.

  2. 설치된 이미지의 디렉토리가 포함된 부모 디렉토리를 탐색합니다.

  3. 설치된 이미지의 디렉토리를 선택합니다.

  4. 열기를 누릅니다.

    업데이트 도구는 사용자가 선택한 디렉토리에 IPS 사용이 가능한 설치된 이미지가 포함되어 있는 것을 확인하고 해당 이미지를 응용 프로그램 이미지 창의 이미지 목록에 추가한 다음 새로 추가한 설치된 이미지를 선택합니다.
