설치

구성 요소 목록에서 선택한 애드온 또는 업데이트를 설치합니다.

제거

검색

현재 구성 요소 목록의 항목에 대한 정보 속성에서 검색 상자에 입력한 텍스트를 검색합니다. 업데이트 도구는 패키지 이름, 설명, 파일 목록 등을 포함하여 정확하게 일치하는 항목을 모든 속성에서 검색합니다. 정확하게 일치하는 검색 대신 별표(*)를 사용하여 와일드카드 검색을 수행할 수 있습니다.

검색을 취소하려면 검색 상자의 오른쪽에 있는 X를 누릅니다. 최근 검색을 보고 선택하려면 검색 상자의 왼쪽에 있는 돋보기를 누릅니다.
