데스크탑 알림 프로그램 인터페이스

데스크탑 알림 프로그램이 사용 가능한 업데이트가 있음을 감지하면 데스크탑의 알림 영역에 아이콘을 표시하고 사용자가 소프트웨어 업데이트 도구를 시작하여 사용 가능한 업데이트를 검토 및 적용할 수 있는 창을 표시합니다.

창을 닫을 경우 알림 프로그램 아이콘은 데스크탑의 알림 영역에 남아 있습니다. 아이콘을 누르면 소프트웨어 업데이트 도구가 시작됩니다.

컨텍스트 메뉴를 지원하는 운영 체제의 경우 알림 프로그램 아이콘에서 다음 항목이 포함된 컨텍스트 메뉴를 제공합니다.

메뉴 항목

설명

업데이트 표시 및 적용

설치된 응용 프로그램 이미지에 대한 업데이트를 선택 및 설치할 수 있는 소프트웨어 업데이트 도구를 시작합니다.

업데이트 세부 정보 관리

설치된 응용 프로그램 이미지에서 개별 패키지를 검토 및 선택하여 업데이트를 선택 및 설치할 수 있는 업데이트 도구를 시작합니다.

기본 설정

알림 프로그램에서 업데이트를 확인하는 빈도를 제어할 수 있는 기본 설정 창을 표시합니다.

업데이트 도구 알림 프로그램 정보

업데이트 도구 버전 및 기타 유용한 정보를 나열하는 업데이트 도구 정보 창을 표시합니다.

도움말

업데이트 도구 온라인 도움말 창을 표시합니다.

종료

데스크탑 알림 프로그램을 종료합니다. 다음에 데스크탑에 로그인할 때까지 알림 프로그램이 실행되지 않습니다.

데스크탑 알림 프로그램에 대한 자세한 내용은 온라인 도움말의 "핵심 개념" 절에 있는 데스크탑 알림 프로그램을 참조하십시오.
