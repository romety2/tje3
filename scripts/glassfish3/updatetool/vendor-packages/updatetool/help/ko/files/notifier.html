데스크탑 알림 프로그램

배경 데스크탑 알림 작업(또는 알림 프로그램)이 업데이트 도구와 함께 업데이트를 정기적으로 확인하여 설치된 업데이트 도구에 알려진 모든 설치 이미지에 대한 업데이트를 데스크탑을 통해 알립니다.

알림 프로그램 등록

이미지 초기화 설치 중에 업데이트 자동 확인 및 알림을 사용하도록 선택하면 알림 프로그램이 데스크탑 환경의 시작 프로그램으로 자동 등록됩니다.

초기화 설치 프로그램이 알림 프로그램을 등록하지 않을 경우 업데이트 도구를 1분 이상 실행하기만 하면 알림 프로그램이 자동으로 등록됩니다. 업데이트 도구 기본 설정에서 업데이트 확인 빈도를 "Never" 이외의 값으로 지정하면 알림 프로그램이 자동으로 등록됩니다.

알림 프로그램 시작

알림 프로그램은 사용자가 데스크탑에 로그인할 때마다 백그라운드 프로세스를 시작 및 실행합니다. 알림 프로그램은 하나 이상의 설치 이미지에 대해 사용 가능한 업데이트가 있는 경우에만 데스크탑의 알림 영역에 표시됩니다. (알림 프로그램이 데스크탑 도킹에 표시되지 않는 Mac OS X의 경우는 예외).

업데이트 알림

알림 프로그램이 사용 가능한 업데이트가 있음을 감지하면 데스크탑의 알림 영역에 아이콘을 표시하고 사용자가 소프트웨어 업데이트 도구를 시작하여 사용 가능한 업데이트를 검토 및 적용할 수 있는 창을 표시합니다.

알림 프로그램은 개인 업데이트 도구 설정에서 관리되는 설치 이미지의 목록을 사용하여 업데이트된 구성 요소를 확인하기 위해 어떤 이미지와 저장소를 모니터할 것인지를 결정합니다. 알림 프로그램이 업데이트된 구성 요소를 확인하는 빈도는 업데이트 도구의 기본 설정을 따릅니다. 업데이트 도구의 기본 설정을 통해 알림 프로그램이 업데이트를 확인하는지 여부 및 업데이트 확인 빈도를 수정할 수 있습니다.

편집 -> 기본 설정

알림 프로그램 작업

알림 프로그램을 데스크탑 로그인 시작 프로그램으로 등록하는 데 사용되는 방법은 사용하는 운영체제와 데스크탑 유형에 따라 다릅니다.

  • Windows: 시작 메뉴 -> 프로그램 -> 시작프로그램
  • Gnome: 시스템 -> 기본 설정 -> 세션 -> 시작 프로그램 -> 업데이트 도구 데스크탑 알림 프로그램
  • Mac OS X: 사용자 이름/Library/LaunchAgents/com.sun.ucnotifier.plist

데스크탑에 로그인하면 알림 시작 프로그램이 먼저 사용자 업데이트 도구의 기본 설정에서 관리되는 설치 이미지 목록을 확인하여 어떤 설치 이미지에서 알림 프로그램 작업을 시작할 것인가를 결정합니다. 사용자가 설치 이미지를 설치 및 제거할 때마다 알림 시작 프로그램은 언제나 가장 적합한 설치 이미지를 사용하여 알림 프로그램을 실행하려고 시도합니다.

알림 프로그램 인터페이스

알림 프로그램에서 제공하는 메뉴 옵션에 대한 자세한 내용은 온라인 도움말의 "인터페이스 구성" 절에서 데스크탑 알림 프로그램을 참조하십시오.
