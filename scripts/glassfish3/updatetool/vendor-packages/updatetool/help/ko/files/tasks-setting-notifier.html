알림 프로그램 기본 설정 지정

업데이트 도구의 기본 설정 기능을 통해 알림 프로그램이 업데이트를 확인하는지 여부와 확인하는 빈도에 영향을 미치는 기본 설정을 지정할 수 있습니다.

업데이트 도구를 통해 알림 프로그램 기본 설정 지정

  1. 도우 모음에서 기본 설정을 누릅니다.

    업데이트 도구는 기본 설정 창을 표시합니다.

  2. 업데이트 탭을 누릅니다.

  3. 알림 프로그램이 설치된 구성 요소의 업데이트를 자동으로 확인할 빈도를 선택합니다.

  4. 확인을 누릅니다.

알림 프로그램 메뉴를 통해 알림 프로그램 기본 설정 지정

이 방법은 Mac OS X에서 지원되지 않습니다.

  1. 데스크탑 알림 영역에서 알림 프로그램 아이콘을 마우스 오른쪽 버튼으로 누른 다음 기본 설정을 누릅니다.

    기본 설정 창이 표시됩니다.

  2. 업데이트 탭을 누릅니다.

  3. 알림 프로그램이 설치된 구성 요소의 업데이트를 자동으로 확인할 빈도를 선택합니다.

  4. 확인을 누릅니다.

업데이트 도구 및 알림 프로그램 기본 설정 지정에 대한 자세한 내용은 기본 설정 개요 절을 참조하십시오.
