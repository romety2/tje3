애드온 설치

  1. 응용 프로그램 이미지 창에서 해당 이미지를 검색합니다. 사용 가능한 애드온, 사용 가능한 업데이트, 설치된 구성 요소가 이미지 아래에 나타나지 않을 경우, 이미지 이름 옆의 삼각형 아이콘을 누릅니다.

  2. 이미지 이름 아래에서 사용 가능한 애드온을 누릅니다.

  3. 업데이트 도구가 저장소에 액세스하여 해당 이미지를 확인하고 사용 가능한 애드온의 목록을 새로 고칩니다.

  4. 구성 요소 목록에 사용 가능한 애드온이 있다고 표시되면 설치할 애드온을 선택합니다. 마치면 설치를 눌러 선택한 애드온을 다운로드 및 설치합니다.

사용 가능한 애드온 보기에 표시되는 정보에 대해서는 구성 요소 목록 절에 설명되어 있습니다.

기본적으로 업데이트 도구는 사용 가능한 최신 애드온 구성 요소 버전만 나열합니다. 이전 애드온 버전에 대한 설치를 검토하여 선택하려면 "최신 버전 표시" 기능을 사용합니다.

보기 -> 최신 버전 표시

"기본 게시자"가 사용 가능한 애드온 목록에 미치는 영향에 대한 설명은 저장소 및 게시자 절을 참조하십시오.
