설치된 이미지 새로 만들기

새로운 빈 설치 이미지를 만들고 이를 하나 이상의 구성 요소 저장소에 연결할 경우, 업데이트 도구 GUI의 새 이미지 기능을 사용할 수 있습니다. 빈 설치 이미지를 만든 후에는 저장소에서 애드온 구성 요소를 선택하고 사용자의 새 이미지에 설치할 수 있습니다.

  1. 도구 모음에서 새로 만들기를 누릅니다.

    업데이트 도구에 설치된 이미지 새로 만들기 창이 표시됩니다.

  2. 이미지 제목 상자에 설치 이미지의 이름을 입력합니다.

    이 이름은 응용 프로그램 이미지 창에 설치된 이미지의 이름으로 나타나므로 제목을 비교적 짧게 유지하십시오.

  3. 이미지 디렉토리 상자에 이미지를 저장할 디렉토리의 전체 경로를 입력합니다.

    또는 ...버튼을 누르고 디렉토리를 검색하거나 새로 만듭니다.

    여기에 모든 애드온 구성 요소가 설치됩니다.

  4. 이미지 설명 상자에서 이미지에 대한 설명을 입력합니다(선택사항).

  5. 저장소 영역에서 추가... 버튼을 사용하여 이미지에 하나 이상의 저장소를 추가합니다.

    추가한 각 저장소에 대해 게시자 이름을 지정하고 저장소의 게시자 URL을 입력해야 합니다.

    사용 가능한 애드온 구성 요소와 업데이트의 목록을 볼 때 "소스" 열에 이 게시자 이름이 표시됩니다.

  6. 확인을 누릅니다.

    업데이트 도구는 이미지를 응용 프로그램 이미지 창의 목록에 추가하고 저장소에 액세스하여 설치 가능한 구성 요소의 목록을 수집하고 사용 가능한 애드온 목록에 표시합니다.

설치 이미지에는 복수의 게시자 저장소를 추가할 수 있지만, 이 중 하나는 기본 게시자로 표시해야 합니다. 기본 게시자의 애드온은 설치 이미지에 연결된 다른 게시자를 통해 사용 가능한 같은 이름의 애드온보다 높은 우선 순위를 갖습니다.

기본 게시자와 이 게시자가 구성 요소 설치 및 업데이트에 미치는 영향에 대한 자세한 내용은 저장소 및 게시자 절을 참조하십시오.

이제 새 이미지에 연결된 애드온 구성 요소를 찾아보고 설치할 수 있습니다.
