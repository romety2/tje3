업데이트 알림 시 작업 수행

업데이트 도구에서 업데이트를 자동으로 확인하도록 설정한 경우 업데이트 도구 알림 프로그램은 하나 이상의 업데이트가 있는 것으로 확인되면 데스크탑에 알림 풍선을 표시합니다. 풍선을 누르면 소프트웨어 업데이트 도구가 시작됩니다. 또는 소프트웨어 업데이트 도구가 실행 중인 경우 데스크탑의 전면에 표시됩니다. 설치 이미지 하나에만 업데이트가 있는 경우 소프트웨어 업데이트 도구는 업데이트를 해당 이미지에 적용할지를 묻습니다.

둘 이상의 설치 이미지에 대한 업데이트가 사용 가능한 경우 소프트웨어 업데이트 도구는 업데이트가 있는 이미지를 나열합니다. 대화 상자를 해제한 다음에는 설치할 업데이트를 선택할 수 있습니다.

데스크탑의 알림 영역에 업데이트 도구 알림 프로그램 아이콘이 표시될 때마다 아이콘을 마우스 오른쪽 버튼으로 누르고 "소프트웨어 업데이트 시작"을 눌러 소프트웨어 업데이트 도구를 시작하고 사용 가능한 업데이트를 검토한 후 선택적으로 적용할 수 있습니다. Mac OS X에서는 이 옵션을 사용할 수 없습니다.
