응용 프로그램 이미지 창

응용 프로그램 이미지 창은 업데이트 도구 창 왼쪽에 나타납니다. 이 창에는 업데이트 도구가 관리할 수 있는 설치 이미지가 나열됩니다. 업데이트 도구는 각 설치 이미지에 대해 세가지 목록을 제공합니다.

  • 사용 가능한 애드온: 아직 설치하지 않은 설치 이미지의 구성 요소.

  • 사용 가능한 업데이트: 설치된 구성 요소의 업데이트(즉, 새 버전).

  • 설치된 구성 요소: 설치된 설치 이미지의 구성 요소.

또한 설치 이미지를 마우스 오른쪽 버튼으로 누르면(Macintosh의 경우 Control 키를 누른 채 누르면) 이미지에서 작동하는 명령을 제공하는 컨텍스트 메뉴가 표시됩니다.

설치된 이미지 목록에는 다음과 같은 방식으로 항목이 추가 또는 제거됩니다.

1) 이미지 열기 및 닫기: 업데이트 도구의 이미지 열기... 및 이미지 닫기... 기능을 사용하여 목록에 설치 이미지를 추가하거나 제거할 수 있습니다. 이미지를 열면 자동으로 이미지가 목록에 추가됩니다. 이미지를 닫으면 목록에서 이미지가 제거됩니다.

2) 이미지 내에서 업데이트 도구 사용: 설치 이미지 내에서 업데이트 도구를 시작하면 이미지가 자동으로 목록에 추가됩니다.

설치된 이미지의 목록은 업데이트 도구 사용자 기본 설정의 일부로 유지됩니다. 자세한 내용은 기본 설정 절을 참조하십시오.
