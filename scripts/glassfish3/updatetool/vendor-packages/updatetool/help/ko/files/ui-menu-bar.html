명령

설명

새 이미지...

새 설치 이미지를 생성합니다.

이미지 열기...

기존 설치 이미지를 엽니다.

이미지 닫기

응용 프로그램 이미지 창에서 선택된 이미지를 닫습니다.

이미지 등록 정보...

편집 메뉴

편집 메뉴의 명령:

명령

설명

검색

도구 모음에서 검색 상자를 활성화하여 현재 구성 요소 목록에서 패키지를 검색할 수 있게 해줍니다.

설치

구성 요소 목록에서 선택한 애드온 또는 업데이트를 설치합니다.

제거

구성 요소 목록에서 선택한 설치된 구성 요소를 제거합니다.

모두 선택

구성 요소 목록의 모든 구성 요소를 선택합니다.

모두 선택 취소

구성 요소 목록의 모든 구성 요소를 선택하지 않습니다.

목록 새로 고침

구성 요소 목록의 구성 요소 목록을 새로 고칩니다.

기본 설정...

기본 설정 창을 표시합니다. 업데이트 도구에서 지정할 수 있는 기본 설정에 대한 자세한 내용은 기본 설정을 참조하십시오.

보기 메뉴

보기 메뉴의 명령:

명령

설명

최신 버전 표시

"사용 가능한 애드온" 및 "사용 가능한 업데이트" 보기에서 이미지와 연관된 저장소에서 사용할 수 있는 가장 최신 버전의 패키지를 표시합니다.

기본 설정 창의 보기 탭에서 표시할 최신 버전 수를 제어할 수 있습니다.

도움말 메뉴의 명령:

명령

설명

목차

이 온라인 도움말을 표시합니다.

정보

업데이트 도구 버전 및 기타 유용한 정보를 나열하는 업데이트 도구 정보 창을 표시합니다.
