So erstellen Sie ein neues installiertes Abbild

Wenn Sie ein neues, leeres Installationsabbild erstellen und es einem oder mehreren Komponenten-Repositories zuordnen müssen, können Sie die Funktion Neues Abbild der Update Tool-GUI verwenden. Nach dem Erstellen eines leeren Installationsabbilds können Sie Add-On-Komponenten aus den Repositories auswählen und sie in Ihrem neuen Abbild installieren.

  1. Klicken Sie in der Symbolleiste auf Neu.

    Update Tool zeigt das Fenster Neues Installationsabbild erstellen an.

  2. Geben Sie im Feld Abbildtitel einen Namen für das Installationsabbild ein.

    Da dieser Name im Bereich Installiertes Anwendungsabbild als Name des installierten Abbilds angezeigt wird, sollten Sie versuchen, den Titel verhältnismäßig kurz zu halten.

  3. Geben Sie im Feld Abbildverzeichnis den vollständigen Pfad zu dem Verzeichnis ein, in dem das Abbild gespeichert werden soll.

    Alternativ können Sie auf die Schaltfläche ... klicken und zu dem Verzeichnis wechseln (oder es ggf. erstellen).

    Dies ist der Speicherort, an dem alle Add-On-Komponenten installiert werden.

  4. Geben Sie im Feld Abbildbeschreibung eine optionale Beschreibung des Abbilds ein.

  5. Verwenden Sie im Bereich Repositories die Schaltfläche Hinzufügen..., um ein oder mehrere Repositories zum Abbild hinzuzufügen.

    Sie müssen für jedes Repository, das Sie hinzufügen, einen Publisher-Namen angeben und die URL des Repository eingeben.

    Der Repository-Name wird in der Spalte "Quelle" angezeigt, wenn Sie Listen der verfügbaren Add-On-Komponenten und Updates aufrufen.

  6. Klicken Sie auf OK.

    Update Tools fügt das Abbild zu der Liste im Bereich Installierte Anwendungsabbilder hinzu, kontaktiert die Repositories, erfasst die Liste der installierbaren Komponenten und zeigt die Liste unter Verfügbare Add-Ons an.

Sie können mehrere Publisher-Repositorys zu Ihrem Installationsabbild hinzufügen, doch einer davon muss als bevorzugter Publisher gekennzeichnet sein. Add-Ons aus dem bevorzugten Publisher haben Vorrang gegenüber gleichnamigen Add-Ons, die über andere zu Ihrem Installationsabbild gehörende Publisher verfügbar gemacht werden.

Weitere Informationen zum gewünschten Publisher und seinem Einfluss auf das Installieren und Aktualisieren von Komponenten finden Sie im Abschnitt Repositorys und Publisher.

Nun können Sie Add-Ons durchsuchen und installieren, die Ihrem neuen Abbild zugeordnet sind.
