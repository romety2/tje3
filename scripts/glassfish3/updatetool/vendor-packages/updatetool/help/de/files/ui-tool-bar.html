Die Symbolleiste

Die Symbolleiste enthält folgende Schaltflächen:

Schaltfläche

Beschreibung

Neues Abbild

Erstellt ein neues Installationsabbild.

Abbild öffnen

Öffnet ein bestehendes Installationsabbild.

Einstellungen

Zeigt das Fenster Update Tool: Eigenschaften an Informationen zu den Einstellungen, die Sie in Update Tool festlegen können, finden Sie unter Einstellungen.

Aktualisieren

Aktualisiert die Komponentenliste. Für verfügbare Add-Ons und verfügbare Updates kontaktiert Update Tool kontaktiert das Repository des Installationsabbilds, um die Informationen abzurufen.

Installieren

Installiert die Add-Ons oder Updates, die in der Komponentenliste ausgewählt sind.

Entfernen

Entfernt die installierten Komponenten, die in der Komponentenliste ausgewählt sind.

Suchen

Sucht die Informationsattribute der Elemente in der aktuellen Komponentenliste für den Text, den Sie im Suchfeld eingeben. Updatetool sucht eine genaue Übereinstimmung in allen Attributen, einschließlich Paktename, Beschreibung, Dateiliste usw. Mit dem * (Sternchen) können Sie eine Platzhaltersuche statt einer Suche mit einer genauen Übereinstimmung ausführen.

Klicken Sie auf das X auf der rechten Seite des Suchfelds, um die Suche abzubrechen. Klicken Sie auf das Vergrößerungsglas auf der linken Seite des Suchfelds, um die letzten Suchvorgänge anzuzeigen und auszuwählen.
