Desktop-Meldungsgenerator

Gemeinsam mit Update Tool prüft eine Hintergrund-Benachrichtigungsaufgabe (oder Meldungsgenerator), ob Updates verfügbar sind und meldet Ihnen über den Desktop, ob Updates für alle Installationsabbilder verfügbar sind, die in Ihrer Installation von Update Tool erfasst sind.

Registrieren des Meldungsgenerators

Wenn Sie während der ersten Installation des Abbilds das automatische Prüfen auf Updates und die Benachrichtigung über Updates aktivieren, wird der Meldungsgenerator in Ihrer Desktop-Umgebung automatisch als Startprogramm registriert.

Wenn ein ursprüngliches Installationsprogramm den Meldungsgenerator nicht registriert, können Sie seine automatische Registrierung veranlassen, indem Sie Update Tool mindestens eine Minute lang ausführen. Wenn in den Update Tool-Einstellungen eine andere Prüfhäufigkeit als "Nie" festgelegt ist, wird der Meldungsgenerator automatisch registriert.

Starten des Meldungsgenerators

Bei jeder Anmeldung beim Desktop wird der Meldungsgenerator gestartet und als Hintergrundprozess ausgeführt. Der Meldungsgenerator wird im Benachrichtigungsbereich Ihres Desktops nur angezeigt, wenn für mindestens eines Ihrer Installationsabbilder Updates verfügbar sind. (Eine Ausnahme besteht unter Mac OS X. Hier wird der Meldungsgenerator niemals im Dock Ihres Desktops dargestellt.)

Benachrichtigung über Updates

Wenn der Meldungsgenerator verfügbare Updates ermittelt, werden im Benachrichtigungsbereich Ihres Desktops ein Symbol und ein Fenster angezeigt, über das Sie das Software Update Tool starten können, um die verfügbaren Updates zu prüfen und zu übernehmen.

Der Meldungsgenerator verwendet die Liste der in Ihren persönlichen Update Tool-Einstellungen verwalteten Installationsabbilder, um zu ermitteln, welche Abbilder und Repositories auf aktualisierte Komponenten überwacht werden sollen. In welchen Abständen der Meldungsgenerator nach Updates sucht, richtet sich nach den in Update Tool festgelegten Einstellungen. Sie können in den Einstellungen von Update Tool festlegen, ob der Meldungsgenerator auf das Vorhandensein von Updates prüft und wie häufig diese Prüfungen auf Updates stattfinden.

Verwenden des Meldungsgenerators

Je nach verwendetem Betriebssystem und Desktop-Typ werden unterschiedliche Methoden zur Registrierung des Meldungsgenerators als Startprogramm bei der Desktop-Anmeldung verwendet:

  • Windows: Startmenü -> Programme -> Autostart
  • Gnome: System -> Einstellungen -> Sitzungen -> Startprogramme -> Update Tool-Meldungsgenerator
  • Mac OS X: Benutzername/Library/LaunchAgents/com.sun.ucnotifier.plist

Bei der Anmeldung beim Desktop überprüft ein Meldungsgenerator-Startprogramm zunächst die Liste der Installationsabbilder, die im Einstellungsbereich von Update Tool verwaltet werden, um zu ermitteln, aus welchem Installationsabbild die Meldungsgenerator-Aufgabe gestartet werden soll. Bei der Installation und Deinstallation von Installationsabbildern versucht das Meldungsgenerator-Startprogramm, stets ein gültiges Installationsabbild für die Ausführung des Meldungsgenerators zu verwenden.
