So installieren Sie Add-Ons

  1. Ermitteln Sie im Bereich Anwendungsabbilder das gewünschte Abbild. Wenn unterhalb des Abbilds nicht die Listen Verfügbare Add-Ons, Verfügbare Updates und Installierte Komponenten angezeigt werden, klicken Sie auf das Dreieckssymbol neben dem Abbildnamen.

  2. Klicken Sie unterhalb des Abbildnamens auf Verfügbare Add-Ons.

  3. Update Tool kontaktiert das Repository für das Abbild und aktualisiert die Liste der verfügbaren Add-Ons.

  4. Wenn in der Komponentenliste angezeigt wird, dass Add-Ons vorhanden sind, wählen Sie die Add-Ons aus, die Sie installieren möchten. Klicken Sie abschließend auf "Installieren", um die ausgewählten Add-Ons herunterzuladen und zu installieren.

Die unter Verfügbare Add-Ons angezeigten Informationen werden im Abschnitt Komponentenliste erläutert.

Update Tool führt standardmäßig nur die neueste verfügbare Version der Add-On-Komponenten auf. Über die Funktion "Alle Versionen anzeigen" können Sie eine ältere Version eines Add-Ons prüfen und zur Installation auswählen:

Ansicht -> Alle Versionen anzeigen

Eine Erklärung dazu, wie sich "Bevorzugter Publisher" auf die Liste verfügbarer Add-Ons auswirkt, finden Sie im Abschnitt Repositorys und Publisher.
