Die Komponentenliste

Die Komponentenliste wird auf der rechten Seite des Update Tool-Fensters, unmittelbar unterhalb der Symbolleiste, angezeigt. Sie zeigt verschiedene Informationen an, je nachdem, ob Sie die verfügbaren Add-Ons, die verfügbaren Updates oder die installierten Komponenten eines Installationsabbilds anzeigen.

  • Ansicht Verfügbare Add-Ons

  • Ansicht Verfügbare Updates

  • Ansicht Installierte Komponenten

Wenn Sie außerdem mit der rechten Maustaste auf die Komponentenliste klicken (Steuerung + Klicken auf dem Macintosh), wird ein Kontextmenü angezeigt, in dem Beispiele dargestellt sind, mit denen die vollständige Komponentenliste und die in der Komponentenliste ausgewählten Elemente dargestellt werden.

Ansicht Verfügbare Add-Ons

Spalte

Beschreibung

Komponente

Name des Add-On.

Kategorie

Kategorie des Add-On. Kategorien fassen Softwarekomponenten, die ähnliche Funktionen bieten, zu Gruppen zusammen.

Veröffentlicht

Datum, an dem das Add-On veröffentlicht wurde.

Version

Version des Add-On.

Installationsgröße

Die ungefähre Installationsgröße des Add-On.

Quelle

Der Name des Repositorys, aus dem das Add-On abgerufen wird.

Ansicht Verfügbare Updates

Spalte

Beschreibung

Komponente

Der Name der Komponenten, für die ein Update verfügbar ist.

!

Die Spalte Warnung, in der Add-Ons, die einen Sicherheits-Fix für die installierte Komponente enthalten, mit einem Sternchen (*) markiert sind.

Veröffentlicht

Datum, an dem das Update veröffentlicht wurde.

Neue Version

Die Version der aktualisierten Komponente.

Installierte Version

Die Version der derzeit installierten Komponente.

Herunterladen

Die ungefähre Installationsgröße des Updates. Beachten Sie: Da nur geänderte Dateien in einem Update bereitgestellt werden, ist die Größe eines Updates normalerweise kleiner als die Installationsgröße der Gesamtkomponente.

Der Änderungswert ist 0, wenn keine Dateien, aber andere Attribute, wie Dateiberechtigungen, Komponentenbeschreibungen und andere Metadaten, geändert werden.

Der Änderungswert ist negativ, wenn die Aktualisierung zu einer Entfernung der Dateien oder zu Reduzierungen in der Größe einer oder mehrerer Dateien führt.

Quelle

Der Name des Repositorys, aus dem das Update abgerufen wird.
