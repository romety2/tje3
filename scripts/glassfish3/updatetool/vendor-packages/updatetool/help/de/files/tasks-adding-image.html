So fügen Sie ein bestehendes Abbild zur Liste der installierten Anwendungsabbilder hinzu

In den meisten Fällen werden installierte Abbilder im Rahmen des Herunterladens und der Installation von Software zu Update Tool hinzugefügt. In einigen Situationen können Sie jedoch auch ein installiertes Abbild aus einem anderen System kopieren, um es auf Ihrem System zu verwenden. Nachdem das installierte Abbild auf Ihr System kopiert wurde, müssen Sie es zur Liste der installierten Abbilder hinzufügen, damit Sie Update Tool verwenden können, um das Abbild auf dem neuesten Stand zu halten.

  1. Klicken Sie in der Symbolleiste auf Öffnen.

    Update Tool zeigt das Fenster zur Auswahl des Verzeichnisses für das Installationsabbild an.

  2. Wechseln Sie in das übergeordnete Verzeichnis, das das Verzeichnis des installierten Abbilds enthält.

  3. Wählen Sie das Verzeichnis des installierten Abbilds aus.

  4. Klicken Sie auf Öffnen.

    Update Tool bestätigt, dass das ausgewählte Verzeichnis ein IPS-fähiges installiertes Abbild enthält, fügt das installierte Abbild zur Liste der Abbilder im Bereich Anwendungsabbilder hinzu und wählt das neu hinzugefügte, installierte Abbild aus.
