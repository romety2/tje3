Die Menüleiste

Die Menüleiste enthält drei Menüs:

  • Datei

  • Bearbeiten

  • Ansicht

  • Hilfe

Das Menü Datei

Das Menü Datei enthält folgende Befehle:

Befehl

Beschreibung

Neues Abbild...

Erstellt ein neues Installationsabbild.

Abbild öffnen...

Öffnet ein bestehendes Installationsabbild.

Abbild schließen

Schließen Sie das Abbild, das im Bereich Anwendungsabbilder ausgewählt ist.

Abbildeigenschaften...

Mithilfe dieser Funktion können Sie die Eigenschaften des ausgewählten Installationsabbilds bearbeiten.

Beenden

Beendet Update Tool.

Das Menü Bearbeiten

Das Menü Bearbeiten enthält folgende Befehle:

Befehl

Beschreibung

Suchen

Aktiviert das Feld Suchen in der Symbolleiste, mit dem Sie in den Paketen der aktuellen Komponentenliste suchen können.

Installieren

Installiert die Add-Ons oder Updates, die in der Komponentenliste ausgewählt sind.

Entfernen

Entfernt die installierten Komponenten, die in der Komponentenliste ausgewählt sind.

Alles markieren

Markiert alle Komponenten in der Komponentenliste.

Alle Markierungen aufheben

Hebt die Markierung alles Komponenten in der Komponentenliste auf.

Liste aktualisieren

Aktualisiert die Komponentenliste in der Komponentenliste.

Einstellungen...

Zeigt das Fenster Einstellungen an. Informationen zu den Einstellungen, die Sie in Update Tool festlegen können, finden Sie unter Einstellungen.

Das Menü Ansicht

Das Menü Ansicht enthält folgende Befehle:

Befehl

Beschreibung

Alle Versionen anzeigen

In der Ansicht "Verfügbare Add-Ons" und "Verfügbare Updates" werden die drei neuesten Versionen aller Pakete angezeigt, die in den Repositorys des Abbilds verfügbar sind.

Auf der Registerkarte Ansicht im Fenster Einstellungen können Sie festlegen, wie viele Versionen angezeigt werden.

Das Menü Hilfe

Das Menü Hilfe enthält folgende Befehle:

Befehl

Beschreibung

Inhalt

Zeigt diese Online-Hilfe an.

Info

Zeigt das Fenster "Info zu Update Tool" an, in dem die Update Tool-Version und andere nützliche Informationen angegeben sind.
