Komponenten

Komponenten (auch als Pakete bekannt) sind die installierbaren Einheiten, die von Update Tool und von "pkg", dem Befehlszeilentool von Image Packaging System (IPS), verwaltet werden. Komponenten bestehen aus Dateien, die auf Ihrem System als Einheit innerhalb eines Installationsabbilds installiert sind. Jedes Installationsabbild enthält Informationen, die im Verzeichnis .org.opensolaris,pkg/ verwaltet werden, das die Installation von Komponenten aufzeichnet.

Mit Update Tool können Sie installierte Komponenten hinzufügen, aktualisieren und entfernen. Die Installation einer neuen Komponente, eines so genannten "Add-On", beinhaltet die Übertragung des Inhalts der Komponente und aller zugehörigen Abhängigkeitsdateien aus einem Netzwerk-Repository und die Installation der Komponente unter einem Installationsabbild.

Bei der Aktualisierung einer Komponente werden lediglich die geänderten Dateien aus einem Netzwerk-Repository auf Ihr System übertragen und nur die betroffenen Dateien in einem Installationsabbild aktualisiert. Alle neuen Abhängigkeitsdateien werden ebenfalls im Rahmen des Aktualisierungsprozesses angewendet.

Die Entfernung einer Komponente beinhaltet die Entfernung aller zuvor installierten Dateien und Verzeichnisse, die durch die ursprüngliche Installation und nachfolgende Updates an der Komponente hinzugekommen sind.

Update Tool bietet folgende Ansichten von Komponenten zur Erleichterung des Hinzufügens, Aktualisierens und Entfernens von Komponenten.
