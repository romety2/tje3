Der Bereich Anwendungsabbilder

Der Bereich Anwendungsabbilder wird auf der linken Seite des Update Tool-Fensters angezeigt. Er listet die Installationsabbilder auf, die von Update Tool verwaltet werden können. Für jedes Installationsabbild gibt Update Tool drei Listen an:

  • Verfügbare Add-Ons: Komponenten des Installationsabbilds, die noch nicht installiert wurden.

  • Verfügbare Updates: Updates (also neuere Versionen) von installierten Komponenten.

  • Installierte Komponenten: Komponenten des Installationsabbilds, die installiert wurden.

Wenn Sie außerdem mit der rechten Maustaste auf ein Installationsabbild (Steuerung + Klicken auf dem Macintosh) klicken, wird ein Kontextmenü angezeigt, in dem Befehle zum Steuern des Abbilds dargestellt werden.

Einträge werden auf folgende Weise zu der Liste der installierten Abbilder hinzugefügt bzw. daraus entfernt:

1) Öffnen und Schließen von Abbildern: Sie können Installationsabbilder zur Liste hinzufügen bzw. daraus entfernen, indem Sie die Funktionen Abbild öffnen... und Abbild schließen... von Update Tool verwenden. Beim Öffnen eines Abbilds wird dieses automatisch zur Liste hinzugefügt. Beim Schließen eines Abbilds wird dieses aus der Liste entfernt.

2) Starten von Update Tool innerhalb eines Abbilds: Wenn Sie Update Tool aus einem Installationsabbild heraus starten, wird das Abbild automatisch zur Liste hinzugefügt.

Die Liste der installierten Abbilder wird im Rahmen der Update Tool-Benutzereinstellungen gepflegt. Weitere Informationen finden Sie im Abschnitt Einstellungen.
