Die Registerkarte Abbilddetails

Die Registerkarte Abbilddetails wird immer dann auf der rechten Seite des Update Tool-Fensters angezeigt, wenn ein Installationsabbild ausgewählt wurde, jedoch keine zum Abbild gehörenden Komponenten. Auf dieser Registerkarte werden Details zum Installationsabbild angezeigt.

Oben auf der Registerkarte werden der Titel und eine kurze Beschreibung des Installationsabbilds angezeigt.

Installiertes Abbild – Details anzeigen:

Titel

Beschreibung

Abbildverzeichnis

Der Dateisystempfad des Stammverzeichnisses (Verzeichnis der obersten Ebene) des installierten Abbilds.

Software-Quellen

Die Liste der Software-Repositories, die mit dem Installationsabbild verknüpft sind. Weitere Informationen finden Sie im Abschnitt Repositorys und Publisher.

Über die Schaltfläche Eigenschaften bearbeiten... können Sie die gesamten Einstellungen für das Installationsabbild bearbeiten. Sie können beispielsweise die Liste der Repositories, den Titel und den Text der Beschreibung für das Installationsabbild ändern.
