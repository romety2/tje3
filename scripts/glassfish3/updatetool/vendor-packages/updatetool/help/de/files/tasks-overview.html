Verwenden von Update Tool

Die Aufgaben, die Sie mit Update Tool ausführen, sind in drei Kategorien untergliedert:

  • Häufig ausgeführte Aufgaben:

    • Reagieren auf Update-Benachrichtigungen

    • Prüfen auf Updates

    • Installieren von Add-Ons

    • Prüfen der Installationen installierter Komponenten

  • Gelegentlich ausgeführte Aufgaben:

    • Entfernen von Komponenten

    • Aufheben von Updates

    • Festlegen der Eigenschaften des Meldungsgenerators

    • Ändern der Publisher- und Repository-Einstellungen

  • Selten ausgeführte Aufgaben:

    • Hinzufügen eines bestehenden Abbilds zur Liste der Anwendungsabbilder

    • Erstellen eines neu installierten Abbilds
