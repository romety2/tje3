Die Software-Update-Oberfläche

Das Software-Update-Fenster besteht aus den folgenden Komponenten:

Komponente

Beschreibung

Tabelle Software-Updates

Listet die Installationsabbilder auf, für die Updates verfügbar sind: In der Spalte Größe wird die Größe aller Update-Downloads des Abbilds angezeigt.

Schaltfläche Aktualisieren

Prüft die Remote-Repositorys aller Installationsabbilder auf Updates.

Tabelle Neue Software und Add-Ons

Hier werden neu verfügbare Software und Add-Ons für die Software in den aktuellen Installationsabbildern angezeigt.

Informationsfeld

Hier werden weitere Informationen zu einem Element angezeigt, das Sie in der Tabelle Software-Update oder Neue Software und Add-Ons auswählen.

Kontrollkästchen Lizenzbedingungen

Aktivieren Sie dieses Kontrollkästchen, um die Lizenzbedingungen für die Updates, Software oder Add-Ons zu aktivieren, die Sie installieren möchten. Klicken Sie auf den Link "Lizenzbedingungen", um die Lizenzbedingungen anzuzeigen.

Schaltfläche zum Verwalten von Details

Startet die Update Tool-Oberfläche, mit der Sie Updates auf Paket-nach-Paket-Basis verwalten können.

Schaltfläche Schließen

Beendet das Software Update Tool, ohne Updates oder Installationen auszuführen.

Schaltfläche Installieren

Installiert die Updates, die Software und die ausgewählten Add-Ons.
