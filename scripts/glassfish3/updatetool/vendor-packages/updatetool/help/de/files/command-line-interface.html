Befehlszeilenschnittstelle

Neben Update Tool zur Verwaltung von Installationsabbildern können Sie die Befehlszeilenschnittstelle von Image Packaging System verwenden. Der Befehl "pkg" ist im Verzeichnis /bin eines Installationsabbilds verfügbar.

Die CLI ist besonders nützlich, wenn das relevante System ein Server ist und über keinen Monitor verfügt und wenn Sie Installationsabbilder per Fernzugriff (remote) verwalten.

Praktisch alle Vorgänge zum Hinzufügen, Aktualisieren und Entfernen sowie die Möglichkeit zum Erstellen neuer Installationsabbilder können über die CLI durchgeführt werden.

Weitere Informationen finden Sie in der pkg(1) documentation. Beachten Sie, dass Update Tool und Image Packaging System auf Nicht-OpenSolaris-Plattformen nur die Verwaltung von "Benutzerabbildern" unterstützt, wie in der pkg(1)-Dokumentation referenziert.

Auf der pkg(5) man page finden Sie weitere Hintergrundinformationen zu Image Packaging System.
