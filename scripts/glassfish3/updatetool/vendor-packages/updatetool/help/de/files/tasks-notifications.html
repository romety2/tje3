So reagieren Sie auf Update-Benachrichtigungen

Wenn Sie Update Tool so konfiguriert haben, dass Updates automatisch geprüft werden, zeigt der Update Tool-Meldungsgenerator bei der Ermittlung, ob ein oder mehrere Updates verfügbar sind, eine Meldung auf dem Desktop an. Wenn Sie auf die Meldung klicken, wird das Software Update Tool gestartet oder im Vordergrund des Desktops angezeigt, falls es bereits ausgeführt wird. Wenn nur ein Installationsabbild über Updates verfügt, fragt das Software Update Tool, ob die Updates für das Abbild übernommen werden sollen.

Wenn für mehr als ein Installationsabbild Updates verfügbar sind, stellt das Software Update Tool die Abbilder dar, die über Updates verfügen. Sie können die Updates zur Installation auswählen, nachdem Sie den Dialog geschlossen haben.

Wenn das Symbol des Update Tool-Meldungsgenerators im Benachrichtigungsbereich des Desktops angezeigt wird, können Sie mit der rechten Maustaste auf das Symbol klicken und anschließend auf "Software-Update starten" klicken, um Software Update Tool zu starten und die verfügbaren Updates zu prüfen und wahlweise zu übernehmen. (Diese Option steht nicht für Mac OS X zur Verfügung.)
