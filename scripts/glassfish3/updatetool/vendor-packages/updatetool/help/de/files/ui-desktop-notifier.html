Die Desktop-Meldungsgeneratoroberfläche

Wenn der Meldungsgenerator feststellt, dass Updates verfügbar sind, zeigt er ein Symbol im Benachrichtigungsbereich Ihres Desktops an sowie ein Fenster, über das Sie Update Tool zur Überprüfung und Anwendung der verfügbaren Updates starten können.

Wenn Sie das Fenster schließen, wird das Symbol des Meldungsgenerators weiterhin im Benachrichtigungsbereich des Desktops angezeigt. Wenn Sie auf das Symbol klicken, wird das Software Update Tool angezeigt.

Für Betriebssysteme, die Kontextmenüs unterstützen, bietet das Meldungsgenerator-Symbol ein Kontextmenü mit den folgenden Elementen:

Menüelement

Beschreibung

Updates anzeigen und anwenden

Startet das Software Update Tool, über das Sie Updates für installierte Anwendungsabbilder auswählen und installieren können.

Update-Details verwalten

Startet das Update Tool, über das Sie Updates auswählen und installieren können, indem Sie einzelne Pakete in installierten Anwendungsabbildern prüfen und auswählen.

Einstellungen

Zeigt das Fenster Einstellungen an, in dem Sie festlegen können, wie häufig der Meldungsgenerator auf Updates prüft.

Über den Update Tool-Meldungsgenerator

Zeigt das Fenster "Info zu Update Tool" an, in dem die Update Tool-Version und andere nützliche Informationen angegeben sind.

Hilfe

Zeigt das Fenster mit der Update Tool-Onlinehilfe an.

Beenden

Beendet den Desktop-Meldungsgenerator. Der Meldungsgenerator wird erst ausgeführt, wenn Sie sich wieder auf dem Desktop anmelden.

Weitere Informationen zum Desktop-Meldungsgenerator finden Sie im Abschnitt "Schlüsselkonzepte" der Onlinehilfe unter Desktop-Meldungsgenerator.
