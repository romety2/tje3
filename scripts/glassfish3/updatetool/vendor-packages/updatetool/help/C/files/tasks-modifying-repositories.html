To Modify Repository Settings

Editing Repositories

You can modify the list of repositories associated with an installation image and edit the settings of existing repositories by selecting the image of interest and performing one of the following operations:

  • Right click to display the context menu and select Properties...
  • Select the Edit Properties... button on the Image Details tab.
  • Select File -> Image Properties from the main menu.

Enabled and Disabled Repositories

You can disable a publihser's repository entry without removing it from the list of repositories by marking the checkbox in the Enabled column of the Image Properties. You cannot disable the preferred publisher.

Preferred Publisher

One of the publishers' repositories associated with an image must be marked as the preferred publisher. Add-ons from the preferred publisher will take precedence over the same named add-ons made available through other publishers associated with your installation image.

Refer to the Repositories and Publishers section for more information on the preferred publisher and its impact on installing and updating components.

Certificate-based Access to Repositories

When you need to access a repository that requires SSL-based communication, you can add a client certificate and an associated client key file to a repository entry. Use of an https:// repository URL requires that you provide a client certificate file and a key file. The certificate and key files are in the .pem format and should be provided by the authority who manages the repository.
