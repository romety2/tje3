The Menu Bar

The menu bar contains three menus:

  • File

  • Edit

  • View

  • Help

The File Menu

Commands on the File menu are:

Command

Description

New Image...

Creates a new installation image.

Open Image...

Opens an existing installation image.

Close Image

Closes the image selected in the Application Images pane.

Image Properties...

Enables you to modify the properties of the selected installation image.

Quit

Quits Update Tool.

The Edit Menu

Commands on the Edit menu are:

Command

Description

Search

Activates the Search box in the toolbar, enabling you to search across the packages in the current component list.

Install

Installs the add-ons or updates that are selected in the component list.

Remove

Removes the installed components that are selected in the component list.

Mark All

Marks all components in the component list.

Unmark All

Unmarks all components in the component list.

Refresh List

Refreshes the component list in the component list.

Preferences...

Displays the Preferences window. See Preferences for information about the preferences you can set in Update Tool.

The View Menu

Commands on the View menu are:

Command

Description

Show Recent Versions

In the "Available Add-ons" and "Available Updates" views, shows the three most recent versions of each package available in the repositories associated with the image.

You can control how many recent versions to show on the View tab of the Preferences window.

The Help Menu

Commands on the Help menu are:

Command

Description

Contents

Displays this online help.

About

Displays the About Update Tool window, which lists the Update Tool version and other useful information.
