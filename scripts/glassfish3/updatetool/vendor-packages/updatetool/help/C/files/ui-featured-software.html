The Featured Software Pane

The Featured Software pane appears on the left side of the Update Tool window, below the Application Images pane, when there is featured software associated with the existing application images. It displays two lists of featured software:

  • Applications: Applications that are complementary to applications that are already installed.

  • Add-ons: Especially worthwhile add-ons to applications that are already installed.

When you select Applications, the New Applications list appears on the right side of the Update Tool window just below the toolbar. The list displays the name, version, and download size of each featured application. Below the New Applications list is an area that describes the currently selected new application, provides a Learn More button to learn more about the application, and an Install button to install the application.

When you select Add-ons, the New Add-ons list appears on the right side of the Update Tool window just below the toolbar. The list displays the parent application, name, version, and download size of each featured add-on. Below the New Add-ons list is an area that describes the currently selected new add-on, provides a Learn More button to learn more about the add-on, and an Install button to install the add-on.
