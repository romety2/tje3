The Component List

The component list appears on the right side of the Update Tool window just below the toolbar. It shows different information, depending on whether you are viewing the available add-ons, available updates, or installed components of an installation image.

  • Available Add-ons View

  • Available Updates View

  • Installed Components View

Additionally, if you right-click the component list (Control+click on the Macintosh), a context menu appears, offering commands that operate on the whole component list and items you have selected on the component list.

Available Add-ons View

Column

Description

Component

The name of the add-on.

Category

The category of the add-on. Categories group software components that provide similar functionality.

Published

The date when the add—on was published.

Version

The version of the add-on.

Install Size

The approximate installation size of the add-on.

Source

The name of the repository from where the add-on will be obtained.

Available Updates View

Column

Description

Component

The name of the component for which there is an update available.

!

The alert column, which shows an asterisk (*) for updates that include security fixes to the installed component.

Published

The date when the update was published.

New Version

The version of the updated component.

Installed Version

The version of the currently installed component.

Download

The approximate installation size of the update. Note that since only modified files are delivered within an update, the size of an update is typically less than the installation size of the entire component.

The Change value will be zero when no files will be modified, but other attributes such as file permissions, component descriptions and other metadata will be modified.

The Change value will be negative when the update will result in either removal of files or reductions in the size of one or more files.

Source

The name of the repository from where the update will be obtained.

Column

Description

Category

Version

Installed Size

The approximate installed size of the component.

Source
