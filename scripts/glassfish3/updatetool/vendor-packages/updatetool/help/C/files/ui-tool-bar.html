The Toolbar

Buttons on the toolbar are:

Button

Description

New Image

Creates a new installation image.

Open Image

Opens an existing installation image.

Preferences

Displays the Update Tool: Preferences window. See Preferences for information about the preferences you can set in Update Tool.

Refresh

Refeshes the component list. For available add-ons and available updates, Update Tool contacts the installation image's repository to retrieve the information.

Install

Installs the add-ons or updates that are selected in the component list.

Remove

Removes the installed components that are selected in the component list.

Search

Searches the informational attributes of the items in the current component list for the text you enter in the Search box. Updatetool searches for an exact match across all attributes, including package name, description, file list, and so on. You can use * (asterisk) to perform a wildcard search instead of an exact match search.

Click the X on the right of the Search box to cancel the search; click the magnifying glass on the left of the Search box to view and select recent searches.
