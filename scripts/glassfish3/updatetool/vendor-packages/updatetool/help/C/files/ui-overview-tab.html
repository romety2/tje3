The Overview Tab

The Overview tab appears on the right side of the Update Tool window, below the component list. When a specific component is selected, the Overview tab displays information about the component.

The title and a brief description of the selected component are displayed at the top of the tab.

Selected Component Information:

Title

Description

Package Name

The underlying package name of the component.

Version

The version of the component.

Installed Size

The amount of disk space consumed by the installation of the component.

Date Published

The date and time at which this version of the component was published to the repository (not the date and time at which it was installed in the installation image).

Source

The name of the repository from where the package was either obtained (in the case of installed components) or would be obtained (in the case of add-ons and updates).

Unique Identifier

The fully qualified identifier of this component as it exists in the repository.
