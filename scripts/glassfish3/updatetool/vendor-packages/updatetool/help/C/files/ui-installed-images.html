The Application Images Pane

The Application Images pane appears on the left side of the Update Tool window. It lists the installation images that Update Tool can manage. For each installation image, Update Tool provides three lists:

  • Available Add-ons: Components of the installation image that are not yet installed.

  • Available Updates: Updates (that is, newer versions) to components that are installed.

  • Installed Components: Components of the installation image that are installed.

Additionally, if you right-click an installation image (Control+click on the Macintosh), a context menu appears, offering commands that operate on the image.

Entries are added to and removed from the list of installed images in the following manner:

1) Opening and Closing Images: You can add and remove installation images to and from the list by using the Open Image... and Close Image... features of the Update Tool. Opening an image will automatically add it to the list. Closing an image will remove it from the list.

2) Starting Update Tool within an Image: When you start Update Tool from within an installation image, the image will automatically be added to the list.

The list of installed images is maintained as part of your Update Tool user preferences. Refer to the Preferences section for more information.
