The Desktop Notifier Interface

When the desktop notifier detects that updates are available, it displays an icon in your desktop's notification area and a window from which you can launch the Software Update tool to review and apply the available updates.

If you close the window, the notifier icon remains in your desktop's notification area. Clicking on the icon launches the Software Update tool.

For operating systems that support context menus, the notifier icon provides a context menu containing these items:

Menu Item

Description

Show and Apply Updates

Launches the Software Update tool, from which you can select and install updates for installed application images.

Manage Update Details

Launches the Update Tool, from which you can select and install updates by reviewing and selecting individual packages in installed application images.

Preferences

Displays the Preferences window, from which you can control how often the notifier checks for updates.

About Update Tool Notifier

Displays the About Update Tool window, which lists the Update Tool version and other useful information.

Help

Displays the Update Tool Online Help window.

Exit

Exits the desktop notifier. The notifier will not run until the next time you log in to your desktop.

For more information about the desktop notifier, see Desktop Notifier in the "Key Concepts" section of online help.
