The Software Update Interface

The Software Update window includes these components:

Component

Description

Software Updates table

Lists the installation images for which updates are available. The Size column shows the size of each image's update download.

Refresh button

Checks the remote repositories of all installation images for updates.

New Software and Add-Ons table

Lists newly available software and add-ons related to the software in your current installation images.

Information box

Provides further information about an item you select in the Software Update or New Software and Add-Ons table.

License Terms checkbox

Click this checkbox to accept any license terms required by the updates, software, or add-ons you have select to install. Click the "license terms" link to view the license terms.

Manage Details button

Launches the Update Tool interface, which you can then use to manage updates on a package-by-package basis.

Close button

Exits the Software Update tool without performing any updates or installations.

Install button

Installs the updates, software, or add-ons you have selected.
