Update Tool

The Update Tool enables you to manage software you have installed on your system. You can use Update Tool to perform a variety of tasks; for example:

  • Checking the versions of software currently installed

  • Finding, downloading, and applying updates to software currently installed

  • Finding, downloading, and installing add-ons to software currently installed

Additionally, you can configure Update Tool to check for and notify you of updates on a regular basis.

To learn how to perform these and other tasks, see Using Update Tool.

To learn how Update Tool and IPS (the Image Packaging System) work together to provide tracking and updating features, see Key Concepts.

To learn what the various parts of the Update Tool interface do, see Parts of the User Interface.
