To Create a New Installed Image

When you need to create a new, empty installation image and associate it with one or more repositories of components, you can use the New Image feature of the Update Tool GUI. After you create an empty installation image, you can select add-on components from the repositories and install them into your new image.

  1. On the toolbar, click New.

    Update Tool displays the Create New Installation Image window.

  2. In the Image Title box, enter a name for the installation image.

    Since this name will appear in Application Images pane as the name of the installed image, try to keep the title relatively short.

  3. In the Image Directory box, enter the full path to the directory that will house the image.

    Alternatively, click the ... button and navigate to (and optionally create) the directory.

    This is the location under which all add-on components will be installed.

  4. In the Image Description box, enter an optional description for the image.

  5. In the Repositories area, use the Add... button to add one or more repositories to the image.

    For each repository you add, you must provide a publisher name for the repository and enter the Publisher URL of the repository.

    The publisher name will appear under the "Source" column when you're viewing lists of available add-on components and updates.

  6. Click OK.

    Update Tool adds the image to the list in the Application Images pane, contacts the repositories, gathers the list of installable components, and displays the list under Available Add-ons.

You can add multiple publisher's repositories to your installation image, but one of them must be marked as the preferred publisher. Add-ons from the preferred publisher will take precedence over the same named add-ons made available through other publishers associated with your installation image.

Refer to the Repositories and Publishers section for more information on the preferred publisher and its impact on installing and updating components.

Now you're ready to browse and install add-on components associated with your new image.
