Command-Line Interface

In addition to using the Update Tool to manage installation images, you can use the Image Packaging System command line interface (CLI). The "pkg" command is available under the bin/ directory of an installation image.

The CLI is especially useful when your system of interest is a server and does not have a monitor and when you are remotely accessing systems to manage installation images.

Virtually all of the add, update and remove operations as well as the ability to create new installation images can be performed using the CLI.

Refer to the pkg(1) documentation for further information. Note that Update Tool and Image Packaging System on non-OpenSolaris platforms support management of only "user images" as referenced in the pkg(1) documentation.

The pkg(5) man page provides further background on the Image Packaging System.
