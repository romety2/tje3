元件

元件 (又稱為套裝軟體) 是可安裝的單元，由 Update Tool 與「pkg」管理，pkg 是影像封裝系統 (IPS) 指令行工具。元件由安裝影像內的檔案構成，這些檔案作為一個單元安裝在系統上。每個安裝影像均包含追蹤元件安裝的資訊，這些資訊在目錄 .org.opensolaris,pkg/ 下維護。

使用 Update Tool 可增加、更新和移除已安裝的元件。安裝新元件 (也就是「附加元件」) 會從網路儲存庫中傳送該元件的內容及其所有附屬檔案，並將元件安裝在安裝影像下。

更新元件只會將網路儲存庫中修改過的檔案傳送到您的系統，並且只會更新安裝影像內受影響的檔案。在更新程序中，也會套用所有新的附屬檔案。

移除元件會移除所有先前安裝的檔案與目錄，這些檔案與目錄是由原始安裝與對該元件的後續更新所產生的。

Update Tool 提供以下元件檢視以簡化增加、更新和移除元件的程序：

  • 可用的附加元件
  • 可用的更新
  • 已安裝的元件
