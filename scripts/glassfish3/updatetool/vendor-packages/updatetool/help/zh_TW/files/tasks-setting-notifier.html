設定通知程式喜好設定

透過 Update Tool 的 [喜好設定] 功能，您可以設定可影響通知程式是否檢查更新及其檢查更新之頻率的喜好設定。

透過 Update Tool 設定通知程式喜好設定

  1. 按一下工具列上的 [喜好設定]。

    Update Tool 會顯示 [喜好設定] 視窗。

  2. 按一下 [更新] 標籤。

  3. 選取通知程式自動檢查已安裝元件有無更新的頻率。

  4. 按一下 [確定]。

透過通知程式功能表設定通知程式喜好設定

Mac OS X 中不支援此方法。

  1. 在桌面通知區域中的通知程式圖示上按一下滑鼠右鍵，然後按一下 [喜好設定]。

    即顯示 [喜好設定] 視窗。

  2. 按一下 [更新] 標籤。

  3. 選取通知程式自動檢查已安裝元件有無更新的頻率。

  4. 按一下 [確定]。

請參閱「喜好設定」簡介小節，以取得關於設定 Update Tool 喜好設定和通知程式喜好設定的更多資訊。
