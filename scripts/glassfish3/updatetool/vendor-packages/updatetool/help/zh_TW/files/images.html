安裝影像

Update Tool 可讓您管理軟體應用程式的安裝，這些軟體應用程式包含在系統的單一檔案系統目錄下。總目錄及其包含的所有內容稱為安裝影像。可以在單一系統上維護多個安裝影像。無需特殊的系統管理權限就可以管理安裝的安裝影像。

安裝影像與元件的一或多個網路儲存庫相關。這些儲存庫包含新元件和更新元件，這些元件可套用至安裝影像。

可以組合安裝影像並透過預先安裝的形式提供，然後將其擴充到系統上，也可以由系統上的安裝應用程式建立安裝影像。此外，您還可以使用 Update Tool 的 [新增影像] 功能建立自己的空白安裝影像：

[檔案] -> [新增影像...]

OpenSolaris Image Packaging System 提供了一種基礎架構，透過該基礎架構，Update Tool 可讓您管理安裝影像。pkg(5) 線上手冊提供了更多關於 Image Packaging System 的背景資訊。另請參閱 Image Packaging System 專案網站，以取得更多資訊：

http://opensolaris.org/os/project/pkg
