將現有影像增加至應用程式影像清單

在大多數情況下，下載與安裝軟體會將安裝的影像增加到 Update Tool 中。但是在某些情況下，可以從另一個系統複製已安裝影像以供您的系統使用。將已安裝影像複製到您的系統後，需要將它增加到已安裝影像的清單中，這樣才能使用 Update Tool 使影像保持為最新。

  1. 按一下工具列上的 [開啟]。

    Update Tool 會顯示 [選擇安裝影像目錄] 視窗。

  2. 瀏覽至包含已安裝影像的目錄的父系目錄。

  3. 選取已安裝影像的目錄。

  4. 按一下 [開啟]。

    Update Tool 會確認選取的目錄是否包含啟用 IPS 的已安裝影像，然後將已安裝影像增加至 [應用程式影像] 窗格中的影像清單並選取新增加的已安裝影像。
