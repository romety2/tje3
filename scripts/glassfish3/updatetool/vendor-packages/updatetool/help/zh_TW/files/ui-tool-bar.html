工具列

工具列上的按鈕包括：

按鈕

描述

新增影像

建立新的安裝影像。

開啟影像

開啟現有安裝影像。

喜好設定

顯示 [更新工具：喜好設定] 視窗。請參閱「喜好設定」，以取得關於可在 Update Tool 中設定的喜好設定的資訊。

重新整理

重新整理元件清單。Update Tool 會連絡安裝影像的儲存庫以擷取可用的附加元件和更新的資訊。

安裝

安裝在元件清單中選取的附加元件或更新。

移除

移除在元件清單中選取的已安裝元件。

搜尋

在目前元件清單項目的資訊屬性中搜尋您在 [搜尋] 方塊輸入的文字。Update Tool 會在所有屬性 (包括套裝軟體名稱、描述、檔案清單等等) 中搜尋精確符合項。您可以使用 * (星號) 來執行萬用字元搜尋代替精確符合項搜尋。

按一下 [搜尋] 方塊右側的 X 可以取消搜尋，按一下 [搜尋] 方塊左側的放大鏡可以檢視和選取最近的搜尋。
