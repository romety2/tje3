設定網路代理伺服器喜好設定

您可以設定影響 Update Tool 和桌面通知程式連線到網際網路之方式的喜好設定。

透過 Update Tool 設定網路代理伺服器喜好設定

  1. 按一下工具列上的 [喜好設定]。

    Update Tool 會顯示 [喜好設定] 視窗。

  2. 如果您直接連線到網際網路，請按一下 [直接連線到網際網路]。

    如果您透過代理伺服器連線到網際網路，請按一下 [手動代理伺服器配置]。然後，提供您透過其連線的代理伺服器的資訊：

    • [HTTP 代理伺服器]：輸入代理伺服器的完整名稱。
    • [連接埠]：輸入代理伺服器上您透過其連線的連接埠號碼。
    • [代理伺服器需要認證]：如果代理伺服器要求您登入，請標記此選項並填寫 [使用者名稱] 與 [密碼] 欄位；否則，請取消標記此選項。
    • [使用者名稱]：輸入您用來登入代理伺服器的使用者名稱。
    • [密碼]：輸入您用來登入代理伺服器的密碼。

  3. 按一下 [確定]。

透過通知程式功能表設定網路代理伺服器喜好設定

Mac OS X 中不支援此方法。

  1. 在桌面通知區域中的通知程式圖示上按一下滑鼠右鍵，然後按一下 [喜好設定]。

    即顯示 [喜好設定] 視窗。

  2. 按照上述方法繼續作業。
