功能表列

功能表列包含四個功能表：

  • 檔案

  • 編輯

  • 檢視

  • 說明

[檔案] 功能表

[檔案] 功能表的指令包括：

指令

描述

新增影像...

建立新的安裝影像。

開啟影像...

開啟現有安裝影像。

關閉影像

關閉 [應用程式影像] 窗格中已選取的影像。

影像特性...

可讓您修改已選取的安裝影像的特性。

退出

退出 Update Tool。

[編輯] 功能表

[編輯] 功能表的指令包括：

指令

描述

搜尋

在工具列中啟動 [搜尋] 方塊，將使您能夠在目前元件清單中的套裝軟體中進行搜尋。

安裝

安裝在元件清單中選取的附加元件或更新。

移除

移除在元件清單中選取的已安裝元件。

全部標記

標記元件清單中的所有元件。

全部取消標記

取消標記元件清單中的所有元件。

重新整理清單

重新整理元件清單中的元件清單。

喜好設定...

顯示 [喜好設定] 視窗。請參閱「喜好設定」，以取得關於可在 Update Tool 中設定的喜好設定的資訊。

[檢視] 功能表

[檢視] 功能表的指令包括：

指令

描述

顯示最近的版本

在 [可用的附加元件] 檢視和 [可用的更新] 檢視中，會顯示與影像相關聯的儲存庫所提供的每個套裝軟體最新的三個版本。

您可以控制在 [喜好設定] 視窗的 [檢視] 標籤上顯示的最近版本的數量。

[說明] 功能表

[說明] 功能表的指令包括：

指令

描述

內容

顯示此線上說明。

關於

顯示 [關於更新工具] 視窗，其中列出 Update Tool 版本和其他有用資訊。
