執行更新通知

如果您已將 Update Tool 設定為自動檢查更新，則當 Update Tool 通知程式確定有一個或多個可用的更新時，就會在桌面上顯示一個通知說話方框。按一下該說話方框，將啟動 Software Update 工具，或者，如果 Software Update 工具已在執行中，則會將其顯示在桌面上。如果只有一個安裝影像有更新，Software Update 工具會詢問您是否要將更新套用至該影像。

如果多個安裝影像有更新，Software Update 工具會列出有更新的影像。關閉該對話方塊後，您便可以選取更新以進行安裝。

只要 Update Tool 通知程式圖示顯示在桌面的通知區域中，您便可以在該圖示上按一下滑鼠右鍵，然後按一下 [啟動 Software Update] 來啟動 Software Update 工具，即可查看並選擇性地套用可用的更新。(Mac OS X 中沒有此選項)。
