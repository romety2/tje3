桌面通知程式介面

當桌面通知程式偵測到有可用的更新時，會在桌面的通知區域顯示一個圖示並會顯示一個視窗，您可以從該視窗啟動 Software Update 工具，以查看和套用可用的更新。

如果您關閉該視窗，該通知程式圖示會保持在桌面的通知區域中。按一下該圖示可啟動 Software Update 工具。

對於支援右鍵功能表的作業系統，該通知程式圖示會提供包含以下項目的右鍵功能表：

功能表項目

描述

顯示和套用更新

啟動 Software Update 工具，從中您可以選取並安裝已安裝應用程式影像的更新。

管理更新詳細資訊

啟動 Update Tool，從中您可以透過查看並選取已安裝應用程式影像中的各個套裝軟體來選取和安裝更新。

喜好設定

顯示 [喜好設定] 視窗，從中您可以控制通知程式檢查更新的頻率。

關於 Update Tool 通知程式

顯示 [關於更新工具] 視窗，其中列出 Update Tool 版本和其他有用資訊。

說明

顯示 [Update Tool 線上說明] 視窗。

結束

結束桌面通知程式。在下次登入桌面之前，通知程式將不再執行。

如需有關桌面通知程式的更多資訊，請參閱線上說明的「重要概念」一節中的「桌面通知程式」。
