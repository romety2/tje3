元件清單

元件清單顯示在 [Update Tool] 視窗的右側，工具列的正下方。它會顯示不同的資訊，這取決於檢視的是安裝影像的可用附加元件、可用更新還是已安裝元件。

  • [可用的附加元件] 檢視

  • [可用的更新] 檢視

  • [已安裝的元件] 檢視

此外，如果您在元件清單上按一下滑鼠右鍵 (在 Macintosh 上則是於按住 Control 的同時再按一下)，系統將顯示一個右鍵功能表，提供可對整個元件清單及您已在元件清單上選取的項目執行的指令。

[可用的附加元件] 檢視

欄

描述

元件

附加元件的名稱。

種類

附加元件的種類。將提供類似功能的軟體元件分類到同一組。

已發行

發行附加元件的日期。

版本

附加元件的版本。

安裝大小

附加元件的近似安裝大小。

來源

要從中取得附加元件的儲存庫的名稱。

[可用的更新] 檢視

欄

描述

元件

有可用更新的元件的名稱。

!

警示欄，如果更新包含已安裝元件的安全性修正，將顯示星號 (*)。

已發行

發行更新的日期。

新版本

更新元件的版本。

已安裝的版本

目前已安裝的元件的版本。

下載

更新的近似安裝大小。注意，因為更新中只提供已修改的檔案，所以更新的大小通常小於整個元件的安裝大小。

如果沒有修改檔案，而是修改了檔案權限、元件描述和其他中介資料等其他屬性，則變更值會為零。

如果更新造成檔案被移除，或是減少了一或多個檔案的大小，則變更值會為負。

來源

要從中取得更新的儲存庫的名稱。

元件的版本。

安裝的大小

來源
