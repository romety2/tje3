建立新的安裝影像

需要新建空白安裝影像並將它與元件的一或多個儲存庫相關聯時，可以使用 Update Tool GUI 的 [新增影像] 功能。建立空白安裝影像後，可以選取儲存庫中的附加元件，並將它們安裝到新影像中。

  1. 按一下工具列上的 [新增]。

    Update Tool 會顯示 [建立新的安裝影像] 視窗。

  2. 在 [影像標題] 方塊中，輸入安裝影像的名稱。

    由於此名稱會在 [應用程式影像] 窗格中顯示為已安裝影像的名稱，因此請儘量使用較短的標題。

  3. 在 [影像目錄] 方塊中，輸入要儲存影像的目錄的完整路徑。

    也可按一下 [...] 按鈕並瀏覽至目錄 (或者選擇建立目錄)。

    這是會安裝所有附加元件的位置。

  4. 在 [影像描述] 方塊中，可輸入影像的選擇性描述。

  5. 在 [儲存庫] 區域中，使用 [增加...] 按鈕將一或多個儲存庫增加至影像中。

    對於增加的每個儲存庫，您必須為其提供發佈者名稱，並輸入它的發佈者 URL。

    檢視可用的附加元件和更新清單時，發佈者名稱會顯示在 [來源] 欄下方。

  6. 按一下 [確定]。

    Update Tool 會將影像增加至 [應用程式影像] 窗格中的清單，然後連絡儲存庫、收集可安裝元件的清單並在 [可用的附加元件] 下顯示該清單。

可以將多個發佈者儲存庫增加到安裝影像中，但其中一個儲存庫必須標記為喜好的發佈者。與安裝影像相關聯的其他發佈者所提供的同名附加元件相比，喜好的發佈者提供的附加元件具有更高的優先順序。

請參閱「儲存庫和發佈者」小節，以取得有關喜好的發佈者及其對元件安裝和更新之影響的更多資訊。

現在您可以瀏覽並安裝附加元件 (與新影像相關聯的附加元件)。
