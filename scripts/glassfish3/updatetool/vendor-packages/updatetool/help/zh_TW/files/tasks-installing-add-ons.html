安裝附加元件

  1. 在 [應用程式影像] 窗格中，找到所需的影像。如果 [可用的附加元件]、[可用的更新] 和 [已安裝的元件] 沒有出現在影像下方，請按一下影像名稱旁的三角形圖示。

  2. 按一下影像名稱下方的 [可用的附加元件]。

  3. Update Tool 會連絡影像的儲存庫，並重新整理可用的附加元件的清單。

  4. 如果元件清單顯示有可用的附加元件，請選取要安裝的附加元件。完成後，請按一下 [安裝] 以下載並安裝選取的附加元件。

[可用的附加元件] 檢視中顯示的資訊在「元件清單」小節中有說明。

依預設，Update Tool 僅會列出附加元件的最新可用版本。如果您要查看並選取舊版附加元件進行安裝，可以使用 [顯示最近的版本] 功能：

[檢視] -> [顯示最近的版本]

「儲存庫和發佈者」小節提供了關於「喜好的發佈者」如何影響可用的附加元件清單的說明。
