指令行介面

除了使用 Update Tool 管理安裝影像外，還可以使用 Image Packaging System 指令行介面 (CLI)。「pkg」指令位於安裝影像的 bin/ 目錄下。

如果所需的系統是伺服器且該系統沒有顯示器，而您正在遠端存取系統以管理安裝影像，則 CLI 特別有用。

實際上，使用 CLI 可以執行所有的增加、更新和移除作業，還能建立新的安裝影像。

請參閱 pkg(1) 文件，以取得更多資訊。請注意，非 OpenSolaris 平台上的 Update Tool 和 Image Packaging System 僅支援「使用者影像」管理，如 pkg(1) 文件中所述。

pkg(5) 線上手冊提供了更多關於 Image Packaging System 的背景資訊。
