Software Update 介面

[Software Update] 視窗包括以下元件：

元件

描述

[軟體更新] 表

列出有更新的安裝影像。[大小] 欄顯示每個影像的更新的下載大小。

[重新整理] 按鈕

檢查所有安裝影像的遠端儲存庫，查看是否有更新。

[新增軟體和附加元件] 表

列出與目前安裝影像中的軟體相關的新可用軟體和附加元件。

[資訊] 方塊

提供有關您在 [軟體更新] 表或 [新增軟體和附加元件] 表中選取的項目的詳細資訊。

[授權規定] 核取方塊

按一下該核取方塊將接受您已選取進行安裝的更新、軟體或附加元件所要求的授權規定。按一下 [授權規定] 連結可檢視授權規定。

[管理詳細資訊] 按鈕

啟動 Update Tool 介面，您可以使用該介面依每個套裝軟體來管理更新。

[關閉] 按鈕

結束 Software Update 工具，而不執行任何更新或安裝。

[安裝] 按鈕

安裝您已選取的更新、軟體或附加元件。
