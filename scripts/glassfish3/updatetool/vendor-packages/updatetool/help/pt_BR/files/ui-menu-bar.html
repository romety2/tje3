A barra de menus

A barra de menus contém três menus:

  • Arquivo

  • Editar

  • Exibir

  • Ajuda

O menu Arquivo

Os comandos do menu Arquivo são:

Comando

Descrição

Nova imagem...

Cria uma nova imagem de instalação

Abrir imagem...

Abre uma imagem de instalação existente.

Fechar imagem

Fecha a imagem selecionada no painel Imagens de aplicativos.

Propriedades da imagem...

Permite modificar as propriedades da imagem de instalação selecionada.

Sair

Sai da Update Tool.

O menu Editar

Os comandos do menu Editar são:

Comando

Descrição

Pesquisar

Ativa a caixa Pesquisar na barra de ferramentas, permitindo pesquisar nos pacotes da lista atual de componentes.

Instalar

Instala os suplementos ou atualizações selecionados na lista de componentes.

Remover

Remove os componentes instalados que estão selecionados na lista de componentes.

Marcar tudo

Marca todos os componentes que se encontram na lista de componentes.

Desmarcar tudo

Desmarca todos os componentes que se encontram na lista de componentes.

Atualizar lista

Atualiza os itens da lista de componentes.

Preferências...

Exibe a janela Preferências. Consulte Preferências para obter informações sobre as preferências que você pode definir na Update Tool.

O menu Exibir

Os comandos do menu Exibir são:

Comando

Descrição

Mostrar versões recentes

Nas exibições "Suplementos disponíveis" e "Atualizações disponíveis", mostra as três versões mais recentes de cada pacote disponíveis nos repositórios associados à imagem.

Você pode controlar a quantidade de versões recentes mostradas na guia Exibir da janela Preferências.

O menu Ajuda

Os comandos do menu Ajuda são:

Comando

Descrição

Conteúdo

Exibe esta ajuda on-line.

Sobre

Exibe a janela Sobre a Update Tool, que lista a versão da Update Tool e outras informações úteis.
