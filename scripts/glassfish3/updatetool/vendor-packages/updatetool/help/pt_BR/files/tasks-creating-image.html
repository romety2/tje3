Para criar uma nova imagem instalada

Quando é necessário criar uma nova imagem vazia de instalação e associá-la a um ou mais repositórios de componentes, use o recurso Nova imagem da interface gráfica do usuário da Update Tool. Após criar a imagem vazia de instalação, você pode selecionar componentes de suplementos nos repositórios e instalá-los na nova imagem.

  1. Na barra de ferramentas, clique em Novo.

    A Update Tool exibe a janela Criar nova imagem de instalação.

  2. Na caixa Título da imagem, digite um nome para a imagem de instalação.

    Esse nome aparecerá no painel Imagens de aplicativos como o nome da imagem instalada; portanto, use um título relativamente breve.

  3. Na caixa Diretório de imagem, digite o caminho completo do diretório que armazenará a imagem.

    Como alternativa, clique no botão ... e navegue até (e crie opcionalmente) o diretório..

    Todos os componentes de suplementos serão instalados nesse local.

  4. Na caixa Descrição da imagem, digite uma descrição opcional da imagem.

  5. Na seção Repositórios, use o botão Adicionar... para acrescentar um ou mais repositórios para a imagem.

    Para cada repositório adicionado, forneça o nome de um editor e especifique o URL do editor do repositório.

    O nome do editor aparecerá na coluna "Origem", quando você exibir listas de componentes de suplementos e atualizações disponíveis.

  6. Clique em OK.

    A Update Tool adiciona a imagem à lista do painel Imagens de aplicativos, entra em contato com os repositórios, coleta a lista de componentes que podem ser instalados e exibe a lista em Suplementos disponíveis..

Você pode adicionar vários repositórios de editor à imagem de instalação, mas um deles deve ser marcado como editor preferencial. Suplementos do editor preferencial têm precedência sobre suplementos de mesmo nome disponibilizados por outros editores associados à imagem de instalação.

Consulte a seção Repositórios e editores, para obter mais informações sobre o editor preferencial e seu impacto na instalação e atualização de componentes.

Agora, você está pronto para procurar e instalar componentes de suplementos associados à nova imagem.
