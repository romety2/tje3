A guia Arquivos

A guia Arquivos aparece no lado direito da janela Update Tool, abaixo da lista de componentes. Quando um componente específico é selecionado, a guia Arquivos exibe uma lista dos arquivos fornecidos pelo componente selecionado.

A lista de arquivos inclui todos os arquivos fornecidos por um componente. Por exemplo, no caso de atualizações, a lista não representa apenas os arquivos modificados.
