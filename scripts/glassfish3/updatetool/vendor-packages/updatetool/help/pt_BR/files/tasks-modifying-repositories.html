Para modificar configurações de repositórios

Edição de repositórios

Para modificar a lista de repositórios associados a uma imagem de instalação e editar as configurações de repositórios existentes, selecione a imagem desejada e execute uma destas operações:

  • Clique com o botão direito do mouse para exibir o menu de contexto e selecione Propriedades...
  • Na guia Detalhes da imagem, selecione o botão Editar propriedades...
  • No menu principal, selecione Arquivo -> Propriedades da imagem.

Repositórios ativados e desativados

Para desativar uma entrada de repositório do editor sem removê-lo da lista de repositórios, marque a caixa de seleção na coluna Ativado das Propriedades da imagem. Não é possível desativar o editor preferencial.

Editor preferencial

Um dos repositórios de editor associados à imagem deve ser marcado como editor preferencial. Suplementos do editor preferencial têm precedência sobre suplementos de mesmo nome disponibilizados por outros editores associados à imagem de instalação.

Consulte a seção Repositórios e editores, para obter mais informações sobre o editor preferencial e seu impacto na instalação e atualização de componentes.

Acesso a repositórios com base em certificados

Quando é necessário acessar um repositório que exige comunicação baseada em SSL, você pode adicionar um certificado de cliente e um arquivo associado de chaves de cliente a uma entrada de repositório. Usar um URL de repositório https:// exige que você forneça um arquivo de certificado de cliente e um arquivo de chaves. O certificado e os arquivos de chaves têm o formato .pem e devem ser fornecidos pela autoridade que gerencia o repositório.
