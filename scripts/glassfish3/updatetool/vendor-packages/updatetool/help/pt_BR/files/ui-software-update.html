A interface Software Update

A janela Software Update inclui estes componentes:

Componente

Descrição

A tabela Atualizações de software

Lista as imagens de instalação para as quais existem atualizações disponíveis. A coluna Tamanho mostra o tamanho do download de cada atualização da imagem.

Botão Atualizar

Verifica a existência de atualizações nos repositórios remotos de todas as imagens de instalação.

Tabela Novos software e suplementos

Lista software e suplementos recentemente disponíveis, relacionados ao software existente nas imagens de instalação atuais.

Caixa Informações

Oferece mais informações sobre o item que você seleciona no Software Update ou na tabela Novos software e suplementos.

Caixa de seleção Termos da licença

Clique nesta caixa de seleção para aceitar quaisquer termos de licença exigidos por atualizações, software ou suplementos selecionados para instalação. Clique no link "termos da licença" para exibir esses termos.

Botão Gerenciar detalhes

Inicia a interface Update Tool, que você pode usar para gerenciar atualizações de cada pacote.

Botão Fechar

Sai da ferramenta Software Update, sem executar atualizações ou instalações.

Botão Instalar

Instala as atualizações, software ou suplementos selecionados.
