Para instalar suplementos

  1. No painel Imagens de aplicativos, localize a imagem desejada. Se Suplementos disponíveis, Atualizações disponíveis e Componentes instalados não aparecerem abaixo da imagem, clique no ícone do triângulo ao lado do nome da imagem.

  2. Abaixo do nome da imagem, clique em Suplementos disponíveis.

  3. A Update Tool entra em contato com o repositório da imagem e atualiza a lista de suplementos disponíveis.

  4. Se a lista de componentes mostrar suplementos disponíveis, selecione os que serão instalados. Quando concluir, clique em Instalar, para baixar e instalar os suplementos selecionados.

As informações exibidas em Suplementos disponíveis são explicadas na seção Lista de componentes.

Por padrão, a Update Tool lista apenas a versão mais recente disponível dos componentes de suplementos. Para examinar e selecionar para instalação uma versão anterior de um suplemento, use o recurso "Mostrar versões recentes":

Exibir -> Mostrar versões recentes

Uma explicação de como o "Editor preferencial" afeta a lista de suplementos disponíveis é fornecida na seçãoRepositórios e editores.
