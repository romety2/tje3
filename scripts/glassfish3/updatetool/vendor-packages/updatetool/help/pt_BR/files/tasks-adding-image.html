Para adicionar uma imagem existente à lista de imagens de aplicativos

Na maioria dos casos, as imagens instaladas são adicionadas à Update Tool como resultado do download e instalação de software. Porém, em algumas situações, você pode copiar uma imagem instalada em outro sistema para usá-la em seu próprio sistema. Depois de copiar a imagem instalada para o seu sistema, é necessário adicioná-la à lista de imagens instaladas, para poder usar a Update Tool e manter a imagem atualizada.

  1. Na barra de ferramentas, clique em Abrir.

    A Update Tool exibe a janela Escolher diretório da imagem de instalação.

  2. Navegue até o diretório pai que contém o diretório da imagem instalada.

  3. Selecione o diretório da imagem instalada.

  4. Clique em Abrir.

    A Update Tool confirma que o diretório selecionado contém uma imagem instalada permitida pelo IPS, adiciona a imagem instalada à lista de imagens do painel Imagens de aplicativos e seleciona a imagem instalada recém-adicionada.
