A guia Descrição

A guia Descrição aparece no lado direito da janela Update Tool, abaixo da lista de componentes. Quando um componente específico é selecionado, a guia Descrição exibe detalhes do componente.

Detalhes do componente selecionado:

Título

Descrição

Descrição

Uma visão geral do componente.

Nome do pacote

O nome do pacote subjacente do componente.

Versão

A versão do componente.

Tamanho instalado

A quantidade de espaço do disco consumida pela instalação do componente.

Data da publicação

A data e a hora em que esta versão do componente foi publicada no repositório (não a data e a hora em que foi instalada na imagem de instalação).

Origem

O nome do repositório em que o pacote foi obtido (no caso de componentes instalados) ou será obtido (no caso de suplementos e atualizações).

Identificador exclusivo

O identificador totalmente qualificado deste componente, conforme ele existe no repositório.
