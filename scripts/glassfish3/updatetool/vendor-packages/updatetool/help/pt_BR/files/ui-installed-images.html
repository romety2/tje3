O painel Imagens de aplicativos

O painel Imagens de aplicativos aparece no lado esquerdo da janela Update Tool. O painel lista as imagens de instalação que a Update Tool pode gerenciar. Para cada imagem de instalação, a Update Tool fornece três listas:

  • Suplementos disponíveis: Componentes da imagem de instalação que ainda não estão instalados.

  • Atualizações disponíveis: Atualizações (ou seja, versões mais recentes) de componentes instalados.

  • Componentes instalados: Componentes da imagem de instalação que estão instalados.

Além disso, se você clicar com o botão direito do mouse em uma imagem de instalação (Control+clique, no Macintosh), um menu de contexto aparecerá, oferecendo comandos a serem executados na imagem.

Entradas são adicionadas e removidas da lista de imagens instaladas, desta maneira:

1) Abrir e fechar imagens: Você pode adicionar e remover imagens de instalação da lista, usando os recursos Abrir imagem... e Fechar imagem... da Update Tool. Abrir uma imagem a adiciona automaticamente à lista. Fechar uma imagem a remove da lista.

2) Iniciar a Update Tool em uma imagem: Quando você inicia a Update Tool em uma imagem de instalação, a imagem é adicionada automaticamente à lista.

A lista de imagens instaladas é mantida como parte das preferências do usuário da Update Tool. Consulte a seção Preferências para obter mais informações.
