Para reverter atualizações

"Reversão" é o processo de restaurar o estado em que um aplicativo se encontrava antes da aplicação de uma atualização. Esse processo consiste em remover as versões de um pacote instaladas como parte da atualização e, em seguida, instalar versões anteriores selecionadas desse pacote.

A forma mais eficaz de se executar a reversão é a utilização da interface de linha de comando (CLI) pkg(5). Porém, antes de executar a reversão, consulte a documentação da atualização para assegurar que todas as personalizações associadas ao componente sejam preservadas após a remoção do componente e sejam reconhecidas durante a reinstalação da versão anterior desejada do componente. Muitos processos de remoção e reinstalação de componentes preservam as personalizações do usuário, mas é recomendável ler a documentação antes de executar o processo de reversão.

Considere também fazer backup de todo o diretório da imagem de instalação. Como os componentes instalados e muitas, senão todas, personalizações associadas se encontram no diretório comum da imagem de instalação, criar uma cópia do diretório da imagem de instalação proporciona uma cópia do estado da instalação anterior ao processo de reversão.

Para executar a reversão, usando a interface de linha de comando (CLI) pkg(5), siga estas etapas.

  1. Use o comando pkg history -l para exibir um log detalhado das atividades de atualização realizadas na imagem do aplicativo.

  2. Examine o histórico do log até encontrar a operação image-update que alterou os pacotes a serem revertidos.

  3. Tome nota dos pacotes que a operação image-update atualizou. Para cada pacote, anote a versão anterior e a nova versão do pacote. Por exemplo, a linha de histórico do log

    pkg:/pkg-java@1.111,0-31.2172:20090608T230741Z -> pkg:/pkg-java@1.111,0-32.2220:20090703T010710Z

    indica que o pacote pkg-java foi atualizado da versão 1.111,0-31.2172 para a versão 1.111,0-32.2220.

  4. Digite um único comando pkg uninstall, especificando todos os pacotes que foram atualizados pela operação image-update. Por exemplo:

    $ pkg uninstall pkg-java

  5. Digite um único comando pkg install, especificando todos os pacotes, inclusive suas versões anteriores, que foram atualizados pela operação image-update. Por exemplo:

    $ pkg install pkg-java@1.111,0-31.2172
