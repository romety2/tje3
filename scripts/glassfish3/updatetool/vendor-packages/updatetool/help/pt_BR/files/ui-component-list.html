A lista de componentes

A lista de componentes aparece no lado direito da janela Update Tool, logo abaixo da barra de ferramentas. Ela mostra informações diferentes, dependendo de você exibir suplementos disponíveis, atualizações disponíveis ou componentes instalados de uma imagem de instalação.

  • Exibição de suplementos disponíveis

  • Exibição de atualizações disponíveis

  • Exibição de componentes instalados

Além disso, se você clicar com o botão direito do mouse na lista de componentes (Control+clique, no Macintosh), um menu de contexto será apresentado, oferecendo comandos que operam em toda a lista de componentes e nos itens selecionados na lista de componentes.

Exibição de suplementos disponíveis

Coluna

Descrição

Componente

O nome do suplemento.

Categoria

A categoria do suplemento. As categorias agrupam componentes de software que propiciam funcionalidades semelhantes.

Publicado

A data de publicação do suplemento.

Versão

A versão do suplemento.

Tamanho da instalação

O tamanho aproximado da instalação do suplemento.

Origem

O nome do repositório em que o suplemento será obtido.

Exibição de atualizações disponíveis

Coluna

Descrição

Componente

O nome do componente para o qual existe uma atualização disponível.

!

A coluna de alerta, que mostra um asterisco (*) para atualizações com correções de segurança do componente instalado.

Publicado

A data de publicação da atualização.

Nova versão

A versão do componente atualizado.

Versão instalada

A versão do componente atualmente instalado.

Download

O tamanho aproximado da instalação da atualização. Apenas arquivos modificados são fornecidos em uma atualização; portanto, o tamanho da atualização costuma ser menor que o tamanho da instalação do componente inteiro.

O valor Alterar será zero quando não houver arquivo a ser modificado, mas outros atributos, como permissões de arquivo, descrições de componentes e outros metadados, serão modificados.

O valor Alterar será negativo, quando a atualização resultar na remoção de arquivos ou na redução do tamanho de um ou mais arquivos.

Origem

O nome do repositório em que a atualização será obtida.
