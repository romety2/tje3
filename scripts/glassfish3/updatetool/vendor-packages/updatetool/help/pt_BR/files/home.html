Update Tool

A Update Tool permite gerenciar o software que você instalou no sistema. Você pode usar a Update Tool para executar várias tarefas; por exemplo:

  • Verificar as versões do software atualmente instalado

  • Localizar, baixar e aplicar atualizações do software atualmente instalado

  • Localizar, baixar e instalar suplementos do software atualmente instalado

Também pode configurar a Update Tool para verificar e avisar regularmente sobre a existência de atualizações.

Para saber como executar essas e outras tarefas, consulte Utilização da Update Tool.

Para saber como a Update Tool e o IPS (Image Packaging System) funcionam em conjunto para propiciar recursos de controle e atualização, consulte Conceitos principais.

Para saber o que fazem os vários componentes da interface da Update Tool, consulte Componentes da interface do usuário.
