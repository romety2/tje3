Componentes da interface do usuário

A interface do usuário da Update Tool apresenta estes componentes:

  • A barra de menus

  • A barra de ferramentas

  • O painel Imagens de aplicativos

  • A guia Detalhes da imagem

  • A Lista de componentes

  • A guia Visão geral

  • A guia Arquivos

  • A guia Dependências

  • A guia Licença

  • O painel Software especial

Além disso, as interfaces do usuário da ferramenta Software Update e do Notificador da área de trabalho proporcionam acesso a recursos especializados.
