Interface de linha de comando

Além de usar a Update Tool ao gerenciar imagens de instalação, você pode usar a interface de linha de comando (CLI) do Image Packaging System (Sistema de gerenciamento de pacotes de imagens). O comando "pkg" está disponível no diretório bin/ da imagem da instalação.

A CLI é bastante útil quando o sistema desejado é um servidor sem monitor e quando você acessa sistemas remotamente para gerenciar imagens de instalação.

Praticamente todas as operações de adicionar, atualizar e remover, além do recurso de criar novas imagens de instalação, podem ser executadas com a CLI.

Consulte a documentação do pkg(1) para obter mais informações. Em plataformas não OpenSolaris, a Update Tool e o Image Packaging System permitem gerenciar apenas "imagens do usuário", conforme indicado na documentação do pkg(1).

A página do manual do pkg(5) oferece mais informações sobre o Image Packaging System.
