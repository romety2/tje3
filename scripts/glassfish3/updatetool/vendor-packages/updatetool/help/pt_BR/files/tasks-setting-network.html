Para definir preferências de proxy de rede

Você pode definir preferências que afetam a maneira como a Update Tool e o notificador da área de trabalho conectam à Internet:

Definição de preferências de proxy de rede na Update Tool

  1. Na barra de ferramentas, clique em Preferências.

    A Update Tool exibe a janela Preferências.

  2. Se você conecta à Internet diretamente, clique em "Conexão direta à Internet".

    Se você conecta à Internet por meio de um servidor proxy, clique em "Configuração manual de proxy". Forneça informações sobre o servidor proxy utilizado para conexão:

    • Proxy HTTP: digite o nome completo do servidor proxy.
    • Porta: digite o número da porta utilizada para conectar ao servidor proxy.
    • Proxy exige autenticação: se o servidor proxy exigir que você faça logon, marque esta opção e preencha os campos Nome de usuário e Senha; caso contrário, desmarque esta opção.
    • Nome de usuário: digite o nome de usuário utilizado para fazer logon no servidor proxy.
    • Senha: digite a senha utilizada para fazer logon no servidor proxy.

  3. Clique em OK.

Definição de preferências de proxy de rede no menu Notificador

Este método não é permitido no Mac OS X.

  1. Clique com o botão direito do mouse no ícone do notificador, na seção de notificação da área de trabalho, e selecione Preferências.

    A janela Preferências é apresentada.

  2. Continue, conforme indicado acima.
