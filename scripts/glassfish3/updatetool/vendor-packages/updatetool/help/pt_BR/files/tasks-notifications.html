Para decidir sobre notificações de atualização

Se você definir a Update Tool para verificar atualizações automaticamente, o Notificador da Update Tool exibirá um balão de notificação na área de trabalho sempre que determinar uma ou mais atualizações disponíveis. Quando você clica no balão, a ferramenta Software Update é iniciada ou, se já está em execução, é colocada em primeiro plano na área de trabalho. Se houver atualizações apenas de uma imagem de instalação. a ferramenta Software Update perguntará se deseja aplicá-las à imagem.

Quando existem atualizações disponíveis de mais de uma imagem de instalação, a ferramenta Software Update lista as imagens que possuem atualizações. Depois de responder à caixa de diálogo, selecione as atualizações para instalação.

Sempre que o ícone Notificador da Update Tool é mostrado na seção de notificação da área de trabalho, você pode clicar com o botão direito do mouse no ícone e selecionar "Iniciar Software Update", para iniciar a ferramenta Software Update e examinar e opcionalmente aplicar as atualizações disponíveis. (No Mac SO X, não existe essa opção).
