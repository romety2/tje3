Interface de mise à jour de logiciel

La fenêtre de mise à jour de logiciel inclut trois composants :

Composant

Description

Tableau de mises à jour de logiciel

Répertorie les images d'installation pour lesquelles des mises à jour sont disponibles. La colonne Taille affiche la taille du téléchargement de chaque mise à jour d'image.

Bouton Actualiser

Recherche les mises à jour de toutes les images d'installation dans les référentiels à distance.

Tableau des nouveaux logiciels et add-on

Répertorie les versions les derniers logiciels et add-on disponibles liés au logiciel de vos images d'installation actuelles.

Zone d'informations

Fournit des informations supplémentaires sur un élément sélectionné dans la mise à jour de logiciel ou dans le tableau des nouveaux logiciels et add-on.

Case à cocher des termes de la licence

Cochez cette case pour accepter les termes de la licence obligatoires pour installer les mises à jour, le logiciel ou les add-on que vous avez sélectionnés. Cliquez sur le lien "Termes de la licence" pour afficher les termes de la licence.

Bouton Gérer les détails

Lance l'interface de l'outil de mise à jour que vous pouvez utiliser afin de gérer les mises à jour pour chaque package.

Bouton Fermer

Permet de quitter l'outil de mise à jour de logiciel sans effectuer aucune mise à jour ou installation.

Bouton Installer

Installe les mises à jour, les logiciels ou les add-on que vous avez sélectionnés.
