Création d'une nouvelle image d'installation

Pour créer une nouvelle image d'installation vide et l'associer à un ou plusieurs référentiels de composants, vous pouvez utilisez la fonctionnalité New Image (Nouvelle image) de l'outil de mise à jour GUI. Une fois l'image d'installation vide, vous pouvez sélectionner des composants add-on dans les référentiels et les installer dans votre nouvelle image.

  1. Dans la barre d'outils, cliquez sur Nouveau.

    L'outil de mise à jour affiche la fenêtre Create New Installation Image (Création d'une nouvelle image d'installation).

  2. Dans la boîte Image Title (Titre de l'image), saisissez un nom pour l'image d'installation.

    Etant donné que ce nom apparaîtra dans la fenêtre Application Images (Images d'applications) comme celui de l'image installée, choisissez un nom relativement court.

  3. Dans la boîte Image Directory (Répertoire de l'image), saisissez le chemin entier vers le répertoire où se trouvera l'image.

    Vous pouvez aussi cliquez sur le bouton ... et naviguer jusqu'au répertoire (ou bien le créer).

    Il s'agit de l'endroit où tous les composants add-on seront installés.

  4. Dans la boîte Image Description (Description de l'image), saisissez une description facultative de l'image.

  5. Dans la zone Référentiels, utilisez le bouton Ajouter... pour ajouter un ou plusieurs référentiels à l'image.

    Pour chaque référentiel ajouté, vous devez lui affecter un nom d'éditeur et saisir l'URL de l'éditeur du référentiel.

    Le nom de l'éditeur apparaîtra sous la colonne "Source" quand vous afficherez les listes des composants add-on et des mises à jour disponibles.

  6. Cliquez sur OK.

    L'outil de mise à jour ajoute l'image à la liste de la fenêtre Images d'applications, contacte les référentiels, réunit la liste des composants à installer et affiche cette liste sous Add-on disponibles.

Vous pouvez ajouter plusieurs référentiels d'éditeur à votre image d'installation mais l'un d'eux doit être marqué comme éditeur préféré. Les add-on de l'éditeur préféré auront priorité sur ceux du même nom provenant d'autres éditeurs associés à votre image d'installation.

Reportez-vous à la section Référentiels et Éditeurs pour de plus amples informations sur l'éditeur préféré et son rôle dans l'installation et la mise à jour de composants.

Vous pouvez maintenant parcourir et installer des composants add-on associés à votre nouvelle image.
