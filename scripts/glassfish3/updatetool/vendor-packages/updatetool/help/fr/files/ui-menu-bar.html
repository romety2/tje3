La barre de menu

La barre de menu comprend trois menus :

  • Fichier

  • Édition

  • Affichage

  • Aide

Le menu Fichier

Les commandes du menu Fichier sont les suivantes :

Commande

Description

New Image... (Nouvelle image...)

Crée une nouvelle image d'installation.

Open Image... (Ouvrir image...)

Ouvre une image d'installation existante.

Close Image (Fermer image)

Ferme l'image sélectionnée dans la fenêtre Application Images (Images d'applications)

Image Properties... (Propriétés de l'image...)

Vous permet de modifier les propriétés de l'image d'installation sélectionnée.

Quitter

Quitte l'outil de mise à jour.

Le menu Édition

Les commandes du menu Édition sont les suivantes :

Commande

Description

Rechercher

Active la zone de recherche de la barre d'outils qui vous permet de rechercher des packages dans la liste de composants actuelle.

Installer

Installe les add-on ou les mises à jour sélectionnées dans la liste des composants.

Supprimer

Supprime les composants installés qui ont été sélectionnés dans la liste de composants.

Mark All (Sélectionner tout)

Sélectionne tous les composants de la liste de composants.

Unmark All (Désélectionner tout)

Désélectionne tous les composants de la liste de composants.

Refresh List (Actualiser la liste)

Actualise la liste des composants.

Préférences...

Affiche la fenêtre Préférences Reportez-vous à Préférences pour plus d'informations sur les préférences à configurer dans l'outil de mise à jour.

Le menu Affichage

Les commandes du menu Affichage sont les suivantes :

Commande

Description

Afficher les versions récentes

"Add-on disponibles" et "Mises à jour disponibles" affichent les trois versions les plus récentes de chaque package disponibles dans les référentiels associés à l'image.

Vous pouvez contrôler le nombre de versions récentes à afficher dans l'onglet Affichage de la fenêtre Préférences.

Le menu Aide

Les commandes du menu Aide sont les suivantes :

Commande

Description

Contents (Contenu)

Affiche l'aide en ligne

À propos de

Affiche la fenêtre À propos de l'outil de mise à jour, qui liste la version de l'outil de mise à jour et d'autres informations utiles.
