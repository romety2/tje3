Installation des Add-on

  1. Dans la fenêtre Application Images (Images d'applications), repérez l'image désirée. Si Add-on disponibles, Mises à jour disponibles et Composants installés ne s'affichent pas sous l'image, cliquez sous le triangle en regard du nom de l'image.

  2. Sous le nom de l'image, cliquez sur Add-on disponibles.

  3. L'outil de mise à jour contacte le référentiel de l'image et actualise la liste des add-on disponibles.

  4. Si la liste des composants indique que des add—ons sont disponibles, sélectionnez ceux que vous souhaitez installer. Une fois cette opération effectuée, cliquez sur Installer pour télécharger et installer les add-on sélectionnés.

Les informations affichées dans la vue Add-on disponibles sont expliquées dans la section Liste des composants.

Par défaut, l'outil de mise à jour n'affiche que la dernière version disponible des composants add-on. Pour revoir d'anciennes versions de add-on et en sélectionner une afin de l'installer, vous pouvez utiliser la fonctionnalité "Afficher les anciennes versions" :

Afficher -> Afficher les versions récentes

Pour une explication du rôle joué par l'éditeur préféré sur la liste des add-on disponibles, reportez-vous à la section Référentiels et Éditeurs.
