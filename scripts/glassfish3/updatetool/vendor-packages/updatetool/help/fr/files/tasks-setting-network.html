Pour configurer les préférences du mandataire du réseau

Vous pouvez configurer des préférences afin de définir la façon dont l'outil de mise à jour et l'avertisseur du bureau se connectent à Internet :

Configuration des préférences du mandataire du réseau au moyen de l'outil de mise à jour

  1. Dans la barre d'outils, cliquez sur Préférences.

    L'outil de mise à jour affiche la fenêtre Préférences.

  2. Pour vous connecter directement à Internet, cliquez sur "Connexion directe à Internet".

    Pour vous connecter à Internet via un serveur mandataire, cliquez sur "Manual proxy configuration" (Configuration manuelle du mandataire) Puis donnez des informations sur le serveur mandataire utilisé pour la connexion :

    • HTTP Proxy : saisissez le nom complet du serveur mandataire.
    • Port : saisissez le numéro de port utilisé pour connecter votre serveur mandataire.
    • Proxy Requires Authentication : si le serveur mandataire exige que vous vous connectiez, cochez cette case et remplissez les champs Nom de l'utilisateur et Mot de passe ; sinon, décochez cette case.
    • Nom de l'utilisateur : saisissez le nom d'utilisateur utilisé pour vous connecter au serveur mandataire.
    • Mot de passe : saisissez le mot de passe utilisé pour vous connecter au serveur mandataire.

  3. Cliquez sur OK.

Configuration des préférences du serveur mandataire via le menu Notifier (Avertisseur)

Cette méthode n'est pas prise en charge sur Mac OS X.

  1. Cliquez avec la touche droite sur l'icône de l'avertisseur dans la zone de notification de votre bureau, puis cliquez sur Préférences.

    Ceci ouvre la fenêtre Préférences.

  2. Continuez comme indiqué précédemment.
