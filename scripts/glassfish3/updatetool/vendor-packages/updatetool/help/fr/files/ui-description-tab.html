L'onglet Description

L'onglet Description apparaît du côté droit de la fenêtre de l'outil de mise à jour, sous la liste des composants. Quand un composant précis est sélectionné, l'onglet Description montre des détails sur ce composant.

Détails sur le composant sélectionné :

Titre

Description

Description

Une présentation du composant.

Nom du package

Le nom du package sous-jacent du composant.

Version

La version du composant.

Taille installée

L'espace occupé sur le disque par l'installation du composant.

Date de publication

La date et l'heure auxquelles cette version du composant a été publiée dans le référentiel (et non la date et l'heure auxquelles elle a été installée dans l'image d'installation).

Source

Le nom du référentiel dont le package provient (dans le cas de composants installé) ou proviendrait (dans le cas d'add-on et de mises à jour).

Identificateur unique

L'identificateur complet de ce composant tel qu'il existe dans le référentiel.
