Configurar las preferencias del notificador

Puede configurar las preferencias relativas a si el notificador comprobará si hay actualizaciones y la frecuencia con la que las comprobará mediante la función Preferencias de Update Tool:

Configuración de las preferencias del notificador mediante Update Tool

  1. En la barra de herramientas, haga clic en Preferencias.

    Update Tool muestra la ventana Preferencias.

  2. Haga clic en la ficha Actualizaciones.

  3. Seleccione la frecuencia con la que el notificador comprobará automáticamente las actualizaciones para los componentes instalados.

  4. Haga clic en Aceptar.

Configuración de las preferencias del notificador mediante el menú del notificador

Este método no es compatible con Mac OS X.

  1. Haga clic con el botón derecho en el icono del notificador del área de notificación del escritorio y, a continuación, haga clic en Preferencias.

    Se mostrará la ventana Preferencias.

  2. Haga clic en la ficha Actualizaciones.

  3. Seleccione la frecuencia con la que el notificador comprobará automáticamente las actualizaciones para los componentes instalados.

  4. Haga clic en Aceptar.

Consulte la sección Preferencias para obtener más información sobre la configuración de las preferencias de Update Tool y del notificador.
