Update Tool

Update Tool permite administrar el software instalado en el sistema. Entre las tareas que podrá realizar con Update Tool se encuentran:

  • Comprobar las versiones del software instalado actualmente

  • Buscar, descargar y aplicar actualizaciones al software que se encuentre instalado

  • Buscar, descargar e instalar complementos en el software que se encuentre instalado

En este sentido, puede configurar Update Tool de tal modo que compruebe y notifique regularmente la existencia de actualizaciones.

Para conocer cómo se realizan estas tareas, entre otras, consulte Uso de Update Tool.

Para conocer cómo interactúan Update Tool e IPS (Image Packaging System) para ofrecer funciones de seguimiento y actualización, consulte Conceptos clave.

Para conocer qué hace cada una de las partes de la interfaz de Update Tool, consulte Partes de la interfaz de usuario.
