Componentes

Los componentes, también llamados paquetes, son las unidades instalables que administran Update Tool y el comando "pkg", la herramienta de línea de comandos Image Packaging System (IPS). Un componente está compuesto por archivos que se instalan como una unidad en el sistema en una imagen de instalación. El directorio .org.opensolaris,pkg/ de las imágenes de instalación registra la información relativa a la instalación de los componentes.

Update Tool permite agregar, actualizar y eliminar componentes. La instalación de un componente nuevo, o "complemento", supone transferir el contenido del componente y sus dependencias de un depósito de red para su instalación en una imagen.

La actualización de un componente supone transferir sólo los archivos que hayan resultado modificados desde un depósito de red a un sistema de destino; de este modo, en la imagen de instalación sólo se modificarán los archivos objeto de la actualización. El proceso de actualización también supone la aplicación de las dependencias nuevas.

Eliminar un componente implica suprimir la totalidad de archivos y directorios creados durante la instalación y las actualizaciones realizadas en los componentes.

Para facilitar los procesos de agregar, actualizar y eliminar componentes, Update Tool dispone de las siguientes vistas:

  • Complementos disponibles
  • Actualizaciones disponibles
  • Componentes instalados
