Configurar las preferencias del servidor proxy de red

Puede configurar las preferencias que afectan al modo en que Update Tool y el notificador de escritorio se conectan a Internet:

Configuración de las preferencias del servidor proxy de red mediante Update Tool

  1. En la barra de herramientas, haga clic en Preferencias.

    Update Tool muestra la ventana Preferencias.

  2. Si se conecta a Internet directamente, haga clic en "Conexión directa a Internet".

    Si se conecta a Internet mediante un servidor proxy, haga clic en "Configuración manual del servidor proxy". A continuación, proporcione información sobre el servidor proxy con el que se conecta:

    • Servidor proxy HTTP: escriba el nombre completo del servidor proxy.
    • Puerto: introduzca el número de puerto mediante el que se conecta en el servidor proxy.
    • El servidor proxy requiere autenticación: si el servidor proxy requiere que inicie sesión, marque esta opción y complete los campos Nombre de usuario y Contraseña; de lo contrario, no marque esta opción.
    • Nombre de usuario: escriba el nombre de usuario que utiliza para iniciar sesión en el servidor proxy.
    • Contraseña: escriba la contraseña que utiliza para iniciar sesión en el servidor proxy.

  3. Haga clic en Aceptar.

Configuración de las preferencias del servidor proxy de red mediante el menú del notificador

Este método no es compatible con Mac OS X.

  1. Haga clic con el botón derecho en el icono del notificador del área de notificación del escritorio y, a continuación, haga clic en Preferencias.

    Se mostrará la ventana Preferencias.

  2. Continúe como anteriormente.
