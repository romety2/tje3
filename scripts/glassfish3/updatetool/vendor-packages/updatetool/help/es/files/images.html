Imágenes de instalación

Update Tool permite administrar instalaciones de aplicaciones de software que estén contenidas en un solo directorio del sistema de archivos. El directorio principal y todo su contenido recibe el nombre de imagen de instalación. En un sistema se pueden mantener varias imágenes de instalación. Para administrar las imágenes de instalación no es necesario contar con privilegios administrativos especiales en el sistema.

Las imágenes de instalación están asociadas a uno o varios depósitos de componentes. Estos depósitos contienen componentes nuevos y actualizados que se pueden aplicar a las imágenes de instalación.

Las imágenes de instalación se pueden ensamblar y entregar en un formato de preinstalación para después ampliarse en el sistema, o pueden crearse a través de una aplicación de instalación en el sistema. Otra posibilidad es utilizar la función "Nueva imagen" de Update Tool para crear su propia imagen de instalación vacía:

Archivo -> Nueva imagen...

Image Packaging System de OpenSolaris ofrece la infraestructura sobre la cual Update Tool permite administrar imágenes de instalación. La página pkg(5) del comando man ofrece información más detallada sobre Image Packaging System. Para obtener más información, también puede consultar el sitio del proyecto Image Packaging System:

http://opensolaris.org/os/project/pkg
