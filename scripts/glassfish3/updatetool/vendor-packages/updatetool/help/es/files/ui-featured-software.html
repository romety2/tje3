Panel Software destacado

El panel Software destacado aparece a la izquierda de la ventana de Update Tool, debajo del panel Imágenes de aplicación, cuando hay disponibles componentes de software destacados asociados a las imágenes de aplicación existentes. Muestra dos listas de software destacado:

  • Aplicaciones: las aplicaciones que complementan a aquellas que ya se encuentran instaladas.

  • Complementos: complemento especialmente útiles para las aplicaciones que ya se encuentran instaladas.

Al seleccionar Aplicaciones, aparece la lista Nuevas aplicaciones a la derecha de la ventana de Update Tool, debajo de la barra de herramientas. En la lista, se muestra el nombre, la versión y el tamaño de descarga de cada aplicación destacada. Debajo de la lista Nuevas aplicaciones, se encuentra un área en la que se describe la nueva aplicación seleccionada actualmente y se proporcionan el botón Más información para obtener más información sobre esa aplicación y el botón Instalar para instalarla.

Al seleccionar Complementos, aparece la lista Nuevos complementos a la derecha de la ventana de Update Tool, debajo de la barra de herramientas. En esta lista, se muestra la aplicación principal, el nombre, la versión y el tamaño de descarga de cada complemento destacado. Debajo de la lista Nuevos complementos, se encuentra un área en la que se describe el nuevo complemento seleccionado actualmente y se proporcionan el botón Más información para obtener más información sobre ese complemento y el botón Instalar para instalarlo.
