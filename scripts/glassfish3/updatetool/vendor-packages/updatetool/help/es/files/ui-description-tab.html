Ficha Descripción

La ficha Descripción aparece en la zona derecha de la ventana de Update Tool, debajo de la lista de componentes. Cuando se selecciona un componente, la ficha Descripción muestra sus detalles.

Detalles del componente seleccionado:

Título

Descripción

Descripción

Una descripción general del componente.

Nombre del paquete

El nombre del paquete subyacente del componente.

Versión

La versión del componente.

Tamaño instalado

La cantidad de espacio en el disco que consume la instalación del componente.

Fecha de publicación

Fecha y hora de publicación de esta versión del componente en el depósito (no la fecha y hora de instalación en la imagen).

Origen

El nombre del depósito de donde se obtuvo el paquete (en el caso de componentes instalados) o se obtendría (en el caso de actualizaciones y complementos).

Identificador único

El identificador totalmente cualificado de este componente tal y como existe en el depósito.
