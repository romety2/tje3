Acciones tras las notificaciones de actualización

Si ha configurado Update Tool para que compruebe automáticamente si hay actualizaciones, el notificador de Update Tool mostrará un globo de notificación en el escritorio cuando determine que hay una o más actualizaciones disponibles. Si hace clic en el globo de texto, se iniciará la herramienta Software Update o, si ya está en ejecución, se mostrará en primer plano en el escritorio. Si sólo una imagen de instalación tiene actualizaciones, Software Update preguntará si desea aplicárselas.

Si hay actualizaciones disponibles para más de una imagen de instalación, la herramienta Software Update enumerará las imágenes para las que hay actualizaciones. Una vez cerrado el cuadro de diálogo, puede seleccionar las actualizaciones que desea instalar.

Siempre que aparezca el icono del notificador de Update Tool en el área de notificación del escritorio, puede hacer clic con el botón derecho en él y, a continuación, en "Iniciar Software Update" para iniciar esta herramienta y comprobar y aplicar opcionalmente las actualizaciones disponibles. (En Mac OS X no dispone de esta opción).
