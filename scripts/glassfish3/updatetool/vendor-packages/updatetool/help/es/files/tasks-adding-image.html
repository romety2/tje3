Agregar una imagen existente a la lista de imágenes de aplicación

En la mayoría de los casos, las imágenes instaladas se agregan a Update Tool tras descargar e instalar una aplicación de software. Pero también puede copiar una imagen instalada en otro sistema para utilizarla en el suyo. Una vez copiada la imagen instalada en el sistema, es necesario agregarla a la lista de imágenes instaladas para permitir que Update Tool pueda encargarse de las consiguientes actualizaciones.

  1. En la barra de herramientas, haga clic en Abrir.

    Update Tool muestra la ventana Elija el directorio de la imagen de instalación.

  2. Vaya al directorio principal que contenga el directorio de la imagen instalada.

  3. Seleccione el directorio de la imagen instalada.

  4. Haga clic en Abrir.

    Update Tool confirma que el directorio seleccionado contiene una imagen instalada habilitada por IPS, agrega la imagen a la lista de imágenes del panel Imágenes de aplicación y selecciona la imagen instalada que se ha agregado recientemente.
