Notificador de escritorio

Una tarea de notificación de escritorio de fondo (o notificador) trabaja junto con Update Tool para comprobar periódicamente si existen actualizaciones o para notificarle actualizaciones, a través de su escritorio, de todas las imágenes de instalación necesarias para la instalación de Update Tool.

Registro del notificador

Si opta por habilitar la comprobación y notificación automática de actualizaciones durante la instalación inicial de su imagen, el notificador se registrará automáticamente como un programa de inicio en el entorno de escritorio.

Si el notificador no se registra tras ejecutar un programa de instalación inicial, puede hacer que se registre automáticamente tan sólo con ejecutar Update Tool durante al menos un minuto. El notificador se registrará automáticamente siempre que las preferencias de Update Tool especifiquen una frecuencia de comprobación de actualizaciones distinta de "Nunca".

Inicio del notificador

Cada vez que se registre en el escritorio, el notificador se iniciará y se ejecutará como proceso de fondo. El notificador sólo aparecerá en el área de notificación del escritorio cuando haya actualizaciones disponibles para una o más de sus imágenes de instalación. (La excepción se da en Mac OS X donde el notificador nunca se representa en la interfaz gráfica de usuario del escritorio).

Notificación de actualizaciones

Si el notificador detecta la disponibilidad de actualizaciones, mostrará un icono en el área de notificación del escritorio junto con una ventana desde la que podrá iniciar la herramienta Software Update para consultar y aplicar las actualizaciones disponibles.

El notificador utiliza la lista de imágenes de instalación administradas especificadas en Update Tool para así determinar qué imágenes y depósitos se deben supervisar para notificar las actualizaciones aplicables. La frecuencia de comprobación de actualizaciones está basada en las preferencias establecidas en Update Tool. Puede modificar si el notificador va a comprobar las actualizaciones de manera exhaustiva y la frecuencia con la que las comprobará mediante las preferencias de Update Tool:

Editar -> Preferencias

Funcionamiento del notificador

El sistema operativo y el tipo de escritorio que utilice determinará el método que se debe seguir para registrar el notificador como programa de inicio tras el registro:

  • Windows: menú Inicio -> Programas -> Inicio
  • Gnome: Sistema -> Preferencias -> Sesiones -> Programas de inicio -> Notificador de escritorio de Update Tool
  • Mac OS X: nombre usuario/Library/LaunchAgents/com.sun.ucnotifier.plist

Tras el registro en el escritorio, un programa de inicio del notificador consulta la lista de las imágenes de instalación incluida en el área de preferencias de Update Tool con el fin de determinar la imagen de instalación desde la que se va a iniciar la tarea del notificador. Cuando instala y desinstala imágenes de instalación, el programa de inicio del notificador intenta siempre utilizar la imagen de instalación más apropiada desde la que poder ejecutar el programa del notificador.

Interfaz del notificador

Para obtener más información acerca de las opciones del menú, consulte Notificador de escritorio en la sección "Partes de la interfaz" de la ayuda en línea.
