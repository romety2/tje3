La barra de herramientas

Los botones de la barra de herramienta son:

Botón

Descripción

Nueva imagen

Crea una imagen de instalación nueva.

Abrir imagen

Abre una imagen de instalación.

Preferencias

Muestra la ventana Update Tool: preferencias Consulte Preferencias para obtener información sobre las preferencias que puede configurar en Update Tool.

Actualizar

Actualiza la lista de componentes. Para obtener actualizaciones y complementos disponibles, Update Tool se pone en contacto con el depósito de la imagen de instalación para recuperar la información.

Instalar

Instala las actualizaciones o los complementos que están seleccionados en la lista de componentes.

Eliminar

Elimina los componentes instalados que están seleccionados en la lista de componentes.

Buscar

Busca el texto introducido en el cuadro de búsqueda en los atributos informativos de los elementos de la lista de componentes actuales. Update Tool busca una coincidencia exacta en todos los atributos, incluidos el nombre del paquete, la descripción, la lista de archivos, etc. Puede utilizar un asterisco (*) para realizar una búsqueda con comodines en lugar de una búsqueda con correspondencias exactas.

Haga clic en el signo X ubicado a la derecha del cuadro de búsqueda para cancelar la búsqueda. Haga clic en el icono de lupa ubicado a la izquierda del cuadro de búsqueda para ver y seleccionar búsquedas recientes.
