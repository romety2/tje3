Interfaz de Software Update

La ventana de Software Update incluye los siguientes componentes:

Componente

Descripción

Tabla de actualizaciones de software

Muestra las imágenes de instalación para las que hay disponibles actualizaciones. La columna Tamaño muestra el tamaño de cada descarga de actualización de la imagen.

Botón Actualizar

Busca actualizaciones en los depósitos remotos de todas las imágenes de instalación.

Tabla de software y complementos nuevos

Muestra los nuevos complementos y aplicaciones disponibles relacionados con el software de las imágenes de instalación actuales.

Cuadro de información

Proporciona información adicional acerca del elemento seleccionado en la tabla de software y complementos nuevos.

Casilla de verificación Condiciones de licencia

Marque esta casilla de verificación para aceptar las condiciones de licencia necesarias para las actualizaciones, el software y los complementos que ha seleccionado para instalar. Haga clic en el vínculo "Condiciones de licencia" para ver las condiciones de la licencia.

Botón Administrar detalles

Inicia la interfaz de Update Tool que puede utilizar para administrar las actualizaciones por paquete individual.

Botón Cerrar

Permite salir de la herramienta Software Update sin realizar ninguna actualización ni instalación.

Botón Instalar

Instala las actualizaciones, el software o los complementos seleccionados.
