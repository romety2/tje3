メニューバー

メニューバーには 3 つのメニューが含まれます。

  • ファイル

  • 編集

  • 表示

  • ヘルプ

「ファイル」メニュー

「ファイル」メニューのコマンドは次のとおりです。

コマンド

説明

新規イメージ...

新しいインストールイメージを作成します。

イメージを開く...

既存のインストールイメージを開きます。

イメージを閉じる

「アプリケーションイメージ」パネルで選択されたイメージを閉じます。

イメージのプロパティー...

選択されたインストールイメージのプロパティーを変更できます。

終了

アップデートツールを終了します。

「編集」メニュー

「編集」メニューのコマンドは次のとおりです。

コマンド

説明

検索

ツールバーの「検索」ボックスを有効にし、現在のコンポーネントリストのパッケージ全体を検索できるようにします。

インストール

コンポーネントのリストで選択されているアドオンまたはアップデートをインストールします。

削除

コンポーネントのリストで選択されているインストール済みのコンポーネントを削除します。

すべて指定

コンポーネントのリスト内にあるすべてのコンポーネントを指定します。

すべての指定を解除

コンポーネントのリスト内にあるすべてのコンポーネントの指定を解除します。

リストの更新

コンポーネントのリスト内にあるコンポーネントのリストを更新します。

設定...

「設定」ウィンドウを表示します。アップデートツールで設定できる設定についての詳細は、「設定」を参照してください。

「表示」メニュー

「表示」メニューのコマンドは次のとおりです。

コマンド

説明

最近のバージョンを表示

「入手可能なアドオン」および「入手可能なアップデート」ビューで、イメージに関連付けられたリポジトリで入手可能なパッケージごとに、最近のものから順に 3 つのバージョンを表示します。

「設定」ウィンドウの「表示」タブに表示する最近のバージョンの数を制御できます。

「ヘルプ」メニュー

「ヘルプ」メニューのコマンドは次のとおりです。

コマンド

説明

内容

このオンラインヘルプを表示します。

バージョン情報

「アップデートツールについて」ウィンドウを表示し、アップデートツールのバージョンおよびその他の役立つ情報を表示します。
