ソフトウェアアップデートインタフェース

「ソフトウェアアップデート」ウィンドウには次のコンポーネントが含まれます。

コンポーネント

説明

「ソフトウェアアップデート」テーブル

アップデートが入手可能なインストールイメージを一覧表示します。「サイズ」列には、ダウンロードする各イメージのアップデートのサイズが表示されます。

「再表示」ボタン

アップデートのすべてのインストールイメージについて、リモートリポジトリを確認します。

「新規ソフトウェアと新規アドオン」テーブル

現在のインストールイメージ内のソフトウェアに関連して、新たに入手可能なソフトウェアとアドオンを一覧表示します。

「情報」ボックス

「ソフトウェアアップデート」または「新規ソフトウェアと新規アドオン」テーブルで選択した項目の詳細を表示します。

「ライセンス条項」チェックボックス

インストールすることを選択したアップデート、ソフトウェア、またはアドオンで必要なライセンス条項に同意するには、このチェックボックスをクリックします。「ライセンス条項」リンクをクリックすると、ライセンス条項が表示されます。

「詳細管理」ボタン

アップデートツールインタフェースを起動します。アップデートツールインタフェースを使用すると、パッケージごとにアップデートを管理できます。

「閉じる」ボタン

アップデートやインストールを実行せずに、ソフトウェアアップデートツールを終了します。

「インストール」ボタン

選択されたアップデート、ソフトウェア、またはアドオンをインストールします。
