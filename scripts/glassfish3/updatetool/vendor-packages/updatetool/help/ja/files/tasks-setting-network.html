ネットワークプロキシ設定を設定する

アップデートツールおよびデスクトップ通知によるインターネットへの接続方法に影響する設定を指定できます。

アップデートツールを介したネットワークプロキシ設定の設定

  1. ツールバーで「設定」をクリックします。

    アップデートツールによって「設定」ウィンドウが表示されます。

  2. インターネットに直接接続する場合は、「インターネットへの直接接続」をクリックします。

    プロキシサーバーを経由してインターネットに接続する場合は、「手動プロキシ設定」をクリックします。続いて、接続で使用するプロキシサーバーについての、次の情報を指定します。

    • HTTP プロキシ: プロキシサーバーの完全な名前を入力します。
    • ポート: プロキシサーバー経由で接続するポート番号を入力します。
    • プロキシに認証が必要: プロキシサーバーへのログインが必要な場合は、このオプションを選択して「ユーザー名」および「パスワード」フィールドに情報を入力します。それ以外の場合は、このオプションを選択せずそのままにします。
    • ユーザー名: プロキシサーバーへのログインに使用するユーザー名を入力します。
    • パスワード: プロキシサーバーへのログインに使用するパスワードを入力します。

  3. 「了解」をクリックします。

通知メニューを使用したネットワークプロキシ設定の設定

この方法は、Mac OS X ではサポートされていません。

  1. デスクトップの通知領域にある通知アイコンを右クリックして、「設定」をクリックします。

    「設定」ウィンドウが表示されます。

  2. 前述の手順に従って操作を継続します。
