デスクトップ通知インタフェース

デスクトップ通知によってアップデートが入手可能であることが検出されると、デスクトップの通知領域にアイコンが表示され、そのウィンドウからソフトウェアアップデートツールを起動して、入手可能なアップデートを確認して適用できます。

ウィンドウを閉じても、通知アイコンはデスクトップの通知領域に引き続き表示されます。アイコンをクリックすると、ソフトウェアアップデートツールが起動します。

コンテキストメニューがサポートされているオペレーティングシステムでは、通知アイコンに次の項目を含むコンテキストメニューが表示されます。

メニュー項目

説明

表示して、アップデートを適用

ソフトウェアアップデートツールを起動します。ソフトウェアアップデートツールからは、インストールされているアプリケーションイメージのアップデートを選択し、インストールすることができます。

アップデートの詳細を管理

アップデートツールを起動します。アップデートツールからは、インストールされているアプリケーションイメージの個々のパッケージを確認して選択することによって、アップデートを選択し、インストールすることができます。

設定

「設定」ウィンドウを表示します。「設定」ウィンドウからは、通知がアップデートを確認する頻度を制御できます。

アップデートツールについて

「アップデートツールについて」ウィンドウを表示し、アップデートツールのバージョンおよびその他の役立つ情報を表示します。

ヘルプ

「アップデートツールオンラインヘルプ」ウィンドウを表示します。

終了

デスクトップ通知を終了します。次回デスクトップにログインするまで、通知は実行されません。

デスクトップ通知については、オンラインヘルプの「重要な概念」セクションの「デスクトップ通知」を参照してください。
