リポジトリの設定を変更する

リポジトリの編集

インストールイメージに関連付けられたリポジトリのリストを変更し、既存のリポジトリの設定の編集を行うには、変更するイメージを選択して、次のいずれかの操作を実行します。

  • 右クリックしてコンテキストメニューを表示し、「プロパティー」を選択します。
  • 「イメージの詳細」タブの「プロパティーを編集...」ボタンを選択します。
  • メインメニューの「ファイル」 -> 「イメージのプロパティー」を選択します。

使用可能なリポジトリと使用不可なリポジトリ

「イメージのプロパティ」の「使用可能」列のチェックボックスにマークを付けることで、発行元のリポジトリのエントリを、リポジトリのリストから削除せずに使用不可にすることができます。優先発行元を使用不可にすることはできません。

優先発行元

イメージに関連付けられた発行元のリポジトリのいずれかを、優先発行元として指定する必要があります。優先発行元からのアドオンは、インストールイメージに関連付けられたその他の発行元経由で入手可能な同じ名前のアドオンよりも優先されます。

優先発行元に関する情報、およびコンポーネントのインストールとアップデートに対する優先発行元の影響については、「リポジトリと発行元」のセクションを参照してください。

リポジトリへの証明書ベースのアクセス

SSL ベースの通信を要求するリポジトリにアクセスする必要がある場合は、クライアント証明書および関連するクライアントの鍵ファイルを、リポジトリのエントリに追加できます。https:// 形式のリポジトリ URL を使用するには、クライアント証明書のファイルと鍵ファイルの指定が必要です。証明書と鍵ファイルは .pem 形式で、リポジトリを管理する機関によって提供されなければなりません。
