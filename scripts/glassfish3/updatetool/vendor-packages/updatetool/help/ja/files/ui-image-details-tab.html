「イメージの詳細」タブ

「イメージの詳細」タブは、インストールイメージが選択されていてそのイメージと関連付けられたコンポーネントが選択されていない場合に、常に「アップデートツール」ウィンドウの右側に表示されます。このタブには、インストールイメージについての詳細が表示されます。

インストールイメージのタイトルと簡単な説明はタブの上部に表示されます。

インストール済みのイメージの表示の詳細:

タイトル

説明

イメージディレクトリ

ルートのファイルシステムパスまたはインストールイメージの最上位ディレクトリ。

ソフトウェアのソース

インストールイメージに関連付けられているソフトウェアリポジトリのリスト。詳細は、「リポジトリと発行元」のセクションを参照してください。

「プロパティーを編集...」ボタンにより、インストールイメージに関連付けられている全般的な設定を変更できます。たとえば、インストールイメージに関連付けられているリポジトリのリスト、タイトル、説明文などを変更できます。
