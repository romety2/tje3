コマンド行インタフェース

アップデートツールを使用してインストールイメージを管理する方法に加えて、Image Packaging System コマンド行インタフェース (CLI) を使用できます。「pkg」コマンドは、インストールイメージの bin/ ディレクトリの下で使用可能です。

操作対象のシステムがサーバーでモニターがない場合、およびシステムにリモートからアクセスしてインストールイメージを管理する場合には、CLI が特に便利です。

CLI を使用すると、新しいインストールイメージの作成だけでなく、追加、アップデート、削除の、事実上すべての操作が可能です。

詳細は、pkg(1) のマニュアルを参照してください。OpenSolaris プラットフォーム以外のアップデートツールおよび Image Packaging System では、pkg(1) のマニュアルで示す「ユーザーイメージ」の管理のみがサポートされます。

pkg(5) のマニュアルページには、Image Packaging System に関する詳細が説明されています。
