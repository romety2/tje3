ツールバー

ツールバー上にあるボタンは次のとおりです。

ボタン

説明

新規イメージ

新しいインストールイメージを作成します。

イメージを開く

既存のインストールイメージを開きます。

設定

アップデートツールの「設定」ウィンドウを開きます。アップデートツールで設定できる設定についての詳細は、「設定」を参照してください。

再表示

コンポーネントのリストを再表示します。入手可能なアドオンと入手可能なアップデートの場合、アップデートツールがインストールイメージのリポジトリに対して問い合わせを行い、情報を収集します。

インストール

コンポーネントのリストで選択されているアドオンまたはアップデートをインストールします。

削除

コンポーネントのリストで選択されているインストール済みのコンポーネントを削除します。

検索

検索ボックスに入力されたテキストの情報属性を、現在のコンポーネントリストの項目から検索します。アップデートツールは、パッケージ名、説明、ファイルリストなど、すべての属性での完全な一致を検索します。完全な一致を検索するのではなく、* (アスタリスク) を使用してワイルドカード検索を実行することもできます。

検索を取り消すには、「検索」ボックスの右側にある「X」をクリックします。最近の検索を表示して選択するには、「検索」ボックスの左側にある検索アイコンをクリックします。
