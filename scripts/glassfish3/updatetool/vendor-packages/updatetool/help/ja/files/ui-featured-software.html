「補足機能ソフトウェア」パネル

「補足機能ソフトウェア」パネルは、既存のアプリケーションイメージに関連する、木野を備えたソフトウェアが存在する場合に、アップデートツールの左側、「アプリケーションイメージ」パネルの下に表示されます。補足機能ソフトウェアの 2 つのリストを表示します:

  • アプリケーション: すでにインストールされたアプリケーションの補足アプリケーション。

  • アドオン: すでにインストールされたアプリケーションの特に役立つアドオン。

アプリケーションを選択すると、新規アプリケーションリストが、アップデートツールの右側、ツールバーの下に表示されます。リストには各補足機能アプリケーションの、名前、バージョン、ダウンロードサイズが表示されます。「新規アプリケーション」リストの下は、現在選択されている新規アプリケーションについて記述されている区画で、アプリケーションの詳細を表示する、詳細ボタンと、アプリケーションをインストールするインストールボタンが表示されます。

アドオンを選択した場合には、新規アドオンリストが、アップデートツールの右側、ツールバーの下に表示されます。リストには各補足機能アドオンの、名前、バージョン、ダウンロードサイズが表示されます。「新規アドオン」リストの下は、現在選択されている新規アドオンについて記述されている区画で、アドオンの詳細を表示する、詳細ボタンと、アドオンをインストールするインストールボタンが表示されます。
