「説明」タブ

「説明」タブは、「アップデートツール」ウィンドウの右側、コンポーネントのリストの下にあります。特定のコンポーネントが選択される際、コンポーネントについての詳細が「説明」タブに表示されます。

選択されたコンポーネントの詳細:

タイトル

説明

説明

コンポーネントの概要

パッケージ名

コンポーネントの配下のパッケージ名

バージョン

コンポーネントのバージョン

インストール後のサイズ

コンポーネントのインストールによって消費されるディスク容量。

公開日

このバージョンのコンポーネントがリポジトリに公開された日付および時間 (インストールイメージにインストールされた日付および時間ではない)。

公開元

パッケージが取得されたリポジトリ (インストール済みのコンポーネントの場合)、またはパッケージが取得されるリポジトリ (アドオンおよびアップデートの場合) の名前。

一意の識別子

リポジトリ内にあるコンポーネントの完全修飾識別子。
