コンポーネントのリスト

コンポーネントのリストは、「アップデートツール」ウィンドウの右側、ツールバーのすぐ下に表示されます。入手可能なアドオン、入手可能なアップデート、またはインストールイメージのインストール済みのコンポーネントなど、表示させている種類によってコンポーネントパネルに表示される情報が異なります。

  • 「入手可能なアドオン」ビュー

  • 「入手可能なアップデート」ビュー

  • 「インストール済みのコンポーネント」ビュー

さらに、コンポーネントのリストを右クリック (Macintosh では Control + クリック) すると、コンテキストメニューが表示され、コンポーネントのリスト全体で動作するコマンドとコンポーネントのリストで選択した項目が表示されます。

インストール済みのバージョン

現在インストール済みのコンポーネントのバージョン。

ダウンロード

アップデートのおおよそのインストールサイズ。アップデートでは変更されたファイルのみが配信されるので、アップデートのサイズは、コンポーネントの全体のインストールサイズよりも通常小さくなります。

変更されたファイルがない場合、「変更」値はゼロになりますが、ファイルのアクセス権、コンポーネントの説明、その他のメタデータなど、その他の属性は変更されます。

アップデートによりファイルが削除あるいは 1 つ以上のファイルのサイズが減少した場合には、「変更」値は負の数になります。

公開元

アップデートが取得されるリポジトリの名前。
