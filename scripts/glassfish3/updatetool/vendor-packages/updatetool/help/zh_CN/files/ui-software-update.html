Software Update 界面

Software Update 窗口包括以下组件：

组件

说明

“软件更新”表

列出有更新可用的安装映像。“大小”列显示了每个映像的更新下载大小。

“刷新”按钮

检查所有安装映像的远程系统信息库有无更新。

“新增软件和附加软件”表

列出与当前安装映像中的软件有关的最新可用软件和附加软件。

“信息”框

提供在“软件更新”或“新增软件和附加软件”表中选择的项目的更多有关信息。

“许可条款”复选框

单击该复选框可接受您已选择安装的更新、软件或附加软件所规定的所有许可条款。单击“许可条款”链接可查看许可条款。

“管理详细信息”按钮

启动 Update Tool 界面，然后您可以使用该界面管理每个软件包的更新。

“关闭”按钮

退出 Software Update 工具，而不执行任何更新或安装。

“安装”按钮

安装您已选择的更新、软件或附加软件。
