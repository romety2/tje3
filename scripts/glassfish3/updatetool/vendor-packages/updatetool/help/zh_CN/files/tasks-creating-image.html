创建新的已安装映像

如果需要创建新的空安装映像并将其与一个或多个组件系统信息库相关联，可使用 Update Tool GUI 的“新建映像”功能。创建空安装映像后，可从系统信息库中选择附加软件组件，并将它们安装到新映像中。

  1. 在工具栏上，单击“新建”。

    Update Tool 会显示“创建新的安装映像”窗口。

  2. 在“映像标题”框中，输入安装映像的名称。

    由于该名称将作为已安装映像的名称显示在“应用程序映像”窗格中，因此该标题应相对简短一些。

  3. 在“映像目录”框中，输入用于放置映像的目录的完整路径。

    或者，单击“...”按钮并导航到（或者创建）目录。

    所有的附加软件组件都将安装在该位置下。

  4. 在“映像说明”框中，输入映像的可选说明。

  5. 在“系统信息库”区域，使用“添加...”按钮向映像添加一个或多个系统信息库。

    对于所添加的每个系统信息库，必须提供系统信息库的发布者名称，并输入系统信息库的发布者 URL。

    查看可用附加软件组件和更新的列表时，发布者名称将出现在“来源”列下方。

  6. 单击“确定”。

    Update Tool 会将映像添加到“应用程序映像”窗格的列表中，联系系统信息库，收集可安装组件的列表，并在“可用附加软件”下显示该列表。

可在安装映像中添加发布者的多个系统信息库，但必须将其中之一标记为首选发布者。首选发布者中的附加软件将优先于通过与安装映像关联的其他发布者提供的同名附加软件。

有关首选发布者及其对安装和更新组件的影响的更多信息，请参见系统信息库和发布者部分。

现在已准备好浏览并安装与新映像相关联的附加软件组件。
