“概述”选项卡

“概述”选项卡显示在 Update Tool 窗口的右侧，组件列表的下方。选中某个特定组件后，“概述”选项卡会显示有关该组件的信息。

选定组件的标题和简要说明显示在该选项卡的顶部。

选定组件的信息：

标题

说明

软件包名称

组件的基础软件包名称。

版本

组件的版本。

安装大小

组件安装所占用的磁盘空间大小。

发布日期

该版本的组件发布到系统信息库的日期和时间（不是在安装映像中安装该组件的日期和时间）。

来源

曾从中获取软件包（对于已安装的组件）或将要从中获取软件包（对于附加软件和更新）的系统信息库的名称。

唯一标识符

组件存在于系统信息库中时的全限定标识符。
