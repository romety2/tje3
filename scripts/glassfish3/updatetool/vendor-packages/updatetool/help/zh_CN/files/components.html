组件

组件（也称为软件包）是可安装单元，由 Update Tool 和·"pkg" 进行管理，"pkg" 是映像包管理系统 (Image Packaging System, IPS) 命令行工具。组件由位于安装映像内的文件组成，这些文件作为一个单元安装在您的系统中。每个安装映像都包含用于跟踪组件安装的信息，这些信息在 .org.opensolaris,pkg/ 目录下维护。

使用 Update Tool，可添加、更新及删除安装的组件。安装新组件（称为“附加软件”）将从网络系统信息库传输组件内容及其所有相关文件，并将该组件安装到安装映像内。

更新组件只会将修改过的文件从网络系统信息库传输到您的系统，并且只更新安装映像中受影响的文件。在更新过程中，还要对所有新的相关文件进行更新。

删除组件将删除在原始安装该组件及后续更新该组件时所引入的所有先前安装的文件和目录。

Update Tool 提供了以下组件视图，以简化添加、更新和删除组件的过程：

  • 可用附加软件
  • 可用更新
  • 已安装的组件
