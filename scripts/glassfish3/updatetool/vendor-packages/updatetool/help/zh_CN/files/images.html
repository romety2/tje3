安装映像

通过 Update Tool 可以管理系统中单个文件系统目录下所包含的各个软件应用程序的安装。总目录及其下面包含的所有内容称为安装映像。可以在单个系统中维护多个安装映像。管理已安装的安装映像不需要特殊的系统管理权限。

安装映像与组件的一个或多个网络系统信息库相关联。这些系统信息库包含可应用于安装映像的新组件和已更新组件。

可以装配安装映像并以预安装的形式交付，然后扩展到系统，也可由系统中的安装应用程序创建。此外，可以使用 Update Tool 的“新建映像”功能创建自己的空安装映像：

文件 -> 新建映像...

OpenSolaris 的映像包管理系统提供了一种基础结构，Update Tool 允许您通过该基础结构管理安装映像。pkg(5) 手册页提供了有关映像包管理系统的更多背景信息。有关更多信息，另请参见映像包管理系统项目站点：

http://opensolaris.org/os/project/pkg
