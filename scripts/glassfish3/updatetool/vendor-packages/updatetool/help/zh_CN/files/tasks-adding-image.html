在应用程序映像列表中添加现有映像

多数情况下，下载和安装软件后，已安装映像便会添加到 Update Tool 中。但在某些情况下，可能会从另一个系统复制已安装映像，以在自己的系统上使用。将已安装映像复制到您的系统后，需将其添加到已安装映像列表中，以便能使用 Update Tool 使映像始终保持最新状态。

  1. 在工具栏上，单击“打开”。

    Update Tool 会显示“选择安装映像目录”窗口。

  2. 导航到包含已安装映像所在目录的父目录。

  3. 选择已安装映像的目录。

  4. 单击“打开”。

    Update Tool 会确认选定的目录是否包含启用 IPS 的已安装映像，并将已安装的映像添加到“应用程序映像”窗格中的映像列表内，然后选择新添加的已安装映像。
