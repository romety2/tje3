命令行界面

除使用 Update Tool 来管理安装映像外，还可以使用映像包管理系统命令行界面 (command line interface, CLI)。在安装映像的 bin/ 目录下提供了 "pkg" 命令。

如果您所负责的系统是一台没有监视器的服务器，并且您要远程访问系统来管理安装映像，则 CLI 尤其有用。

实际上，使用 CLI 能够执行所有的添加、更新和删除操作，也能够创建新的安装映像。

有关更多信息，请参阅 pkg(1) 文档。请注意，非 OpenSolaris 平台上的 Update Tool 和映像包管理系统仅支持对“用户映像”进行管理，如 pkg(1) 文档中所述。

pkg(5) 手册页提供了有关映像包管理系统的更多背景信息。
