安装附加软件

  1. 在“应用程序映像”窗格中，找到所需的映像。如果映像下方未出现“可用附加软件”、“可用更新”和“已安装的组件”，单击映像名称旁的三角形图标。

  2. 在映像名称下方，单击“可用附加软件”。

  3. Update Tool 会联系映像的系统信息库，并刷新可用附加软件的列表。

  4. 如果组件列表显示附加软件可用，则选择要安装的附加软件。完成后，单击“安装”以下载并安装选定的附加软件。

“可用附加软件”视图中显示的信息在组件列表部分中有说明。

默认情况下，Update Tool 只列出附加软件组件的最新可用版本。如果您要查看和选择安装早期版本的附加软件，可以使用“显示最近版本”功能：

视图 -> 显示最近版本
