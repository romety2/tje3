“应用程序映像”窗格

“应用程序映像”窗格显示在 Update Tool 窗口的左侧。其中列出 Update Tool 可管理的安装映像。对于每个安装映像，Update Tool 提供三个列表：

  • 可用附加软件：尚未安装的安装映像组件。

  • 可用更新：已安装组件的更新（即较新的版本）。

  • 已安装的组件：已安装的安装映像的组件。

此外，如果您在安装映像上单击鼠标右键（在 Macintosh 上是按住 Control 键并单击），会显示一个上下文菜单，该菜单中提供了对该映像执行操作的命令。

通过以下方式在已安装映像列表中添加或删除条目：

1) 打开和关闭映像：使用 Update Tool 的“打开映像...”和“关闭映像...”功能可在列表中添加或删除安装映像。打开映像会自动将其添加到列表中。关闭映像会将映像从列表中删除。

2) 启动映像内的 Update Tool：在启动安装映像内的 Update Tool 时，映像会自动添加到列表中。

已安装映像的列表会作为 Update Tool 用户首选项的一部分进行维护。有关更多信息，请参阅首选项部分。
