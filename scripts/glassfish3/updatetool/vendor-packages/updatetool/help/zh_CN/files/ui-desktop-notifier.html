桌面通知程序界面

当桌面通知程序检测到存在可用更新时，它会在您的桌面通知区域显示一个图标，同时还会显示一个窗口，从该窗口中您可以启动 Software Update 工具，以便查看和应用可用的更新。

如果关闭该窗口，通知程序图标仍会保留在桌面的通知区域。单击该图标可启动 Software Update 工具。

对于支持上下文菜单的操作系统，通知程序图标提供了包含以下项目的上下文菜单：

菜单项

说明

显示和应用更新

启动 Software Update 工具，从该工具中，您可以为所安装的应用程序映像选择和安装更新。

管理更新详细信息

启动 Update Tool，从该工具中，您可以通过查看和选择所安装的应用程序映像中的各个软件包选择和安装更新。

首选项

显示“首选项”窗口，从该窗口中，您可以控制通知程序检查更新的频率。

关于 Update Tool 通知程序

显示“关于 Update Tool”窗口，该窗口会列出 Update Tool 版本及其他有用信息。

帮助

显示 Update Tool 联机帮助窗口。

退出

退出桌面通知程序。在您下次登录桌面之前，通知程序不会运行。

有关桌面通知程序的更多信息，请参见联机帮助“重要概念”部分中的桌面通知程序。
