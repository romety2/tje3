菜单栏

菜单栏包含四个菜单：

  • 文件

  • 编辑

  • 视图

  • 帮助

“文件”菜单

“文件”菜单上的命令包括：

命令

说明

新建映像...

创建新的安装映像。

打开映像...

打开现有安装映像。

关闭映像

关闭在“应用程序映像”窗格中选定的映像。

映像属性...

允许您修改选定安装映像的属性。

退出

退出 Update Tool。

“编辑”菜单

“编辑”菜单上的命令包括：

命令

说明

搜索

激活工具栏中的“搜索”框，从而使您可以在当前组件列表中的软件包内执行搜索。

安装

安装在组件列表中选定的附加软件或更新。

删除

删除在组件列表中选定的已安装组件。

标记所有

标记组件列表中的所有组件。

取消标记所有

取消标记组件列表中的所有组件。

刷新列表

刷新组件列表中的组件列表。

首选项...

显示“首选项”窗口。有关可在 Update Tool 中设置的首选项的信息，请参见首选项。

“视图”菜单

“视图”菜单上的命令包括：

命令

说明

显示最近版本

在“可用附加软件”和“可用更新”视图中，显示与映像关联的系统信息库中提供的每个软件包的三个最新版本。

您可以控制在“首选项”窗口的“视图”选项卡中显示的最新版本数。

“帮助”菜单

“帮助”菜单上的命令包括：

命令

说明

目录

显示本联机帮助。

关于

显示“关于 Update Tool”窗口，该窗口会列出 Update Tool 版本及其他有用信息。
