“精选软件”窗格

当存在与现有应用程序映像关联的精选软件时，“精选软件”窗格会显示在 Update Tool 窗口的左侧，“应用程序映像”窗格的下方。该窗格显示了两个精选软件列表：

  • 应用程序：作为已安装应用程序的补充的应用程序。

  • 附加软件：对于已安装的应用程序特别有用的附加软件。

选择“应用程序”时，“新增应用程序”列表会显示在 Update Tool 窗口的右侧，工具栏的下方。该列表显示了每个精选应用程序的名称、版本和下载大小。“新增应用程序”列表下方的区域对当前选定的新应用程序进行了说明，并提供了一个“了解更多”按钮（用于了解有关应用程序的更多信息）和一个“安装”按钮（用于安装应用程序）。

选择“附加软件”时，“新增附加软件”列表会显示在 Update Tool 窗口的右侧，工具栏的下方。该列表显示了每个精选附加软件的父应用程序、名称、版本和下载大小。“新增附加软件”列表下方的区域对当前选定的新附加软件进行了说明，并提供了一个“了解更多”按钮（用于了解有关附加软件的更多信息）和一个“安装”按钮（用于安装附加软件）。
