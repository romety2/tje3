按更新通知操作

如果已将 Update Tool 设置为自动检查更新，则 Update Tool 通知程序在确定存在一个或多个可用更新时会在桌面上显示一个通知气球。如果在该气球上单击，将启动 Software Update 工具，如果该工具已经在运行，则会使其显示在桌面最前方。如果只有一个安装映像有更新，则 Software Update 工具将询问您是否要将更新应用于该映像。

如果有多个安装映像有更新，则 Software Update 工具将列出有更新的映像。退出该对话框后，您可以选择要安装的更新。

只要在桌面的通知区域显示了“Update Tool 通知程序”图标，就可以在该图标上单击鼠标右键，然后单击“启动 Software Update”以启动 Software Update 工具，并查看和应用可用更新（可选）。（Mac OS X 上没有该选项）。
