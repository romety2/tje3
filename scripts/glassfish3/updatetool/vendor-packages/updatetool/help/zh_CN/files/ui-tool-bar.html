工具栏

工具栏上包括以下按钮：

按钮

说明

新建映像

创建新的安装映像。

打开映像

打开现有安装映像。

首选项

显示“Update Tool：首选项”窗口。有关可在 Update Tool 中设置的首选项的信息，请参见首选项。

刷新

刷新组件列表。对于可用附加软件和可用更新，Update Tool 会联系安装映像的系统信息库以检索信息。

安装

安装在组件列表中选定的附加软件或更新。

删除

删除在组件列表中选定的已安装组件。

搜索

在当前组件列表中的项目的信息性属性中搜索您在“搜索”框中输入的文字。Update Tool 会搜索所有属性（包括软件包名称、说明、文件列表等）中的精确匹配项。您可以使用 *（星号）执行通配符搜索，而不是精确匹配项搜索。

单击“搜索”框右侧的 X 可取消搜索，单击“搜索”框左侧的放大镜可查看和选择最近的搜索。
