设置网络代理首选项

您可以设置影响 Update Tool 以及桌面通知程序与 Internet 的连接方式的首选项：

通过 Update Tool 设置网络代理首选项

  1. 在工具栏上，单击“首选项”。

    Update Tool 将显示“首选项”窗口。

  2. 如果要直接连接到 Internet，请单击“直接连接到 Internet”。

    如果要通过代理服务器连接到 Internet，请单击“手动代理配置”。然后，提供连接所使用的代理服务器的信息：

    • HTTP 代理：输入代理服务器的全称。
    • 端口：输入代理服务器上连接所使用的端口号。
    • 代理需要验证：如果代理服务器要求您登录，则标记该选项并填写“用户名”和“密码”字段，否则，请取消标记该选项。
    • 用户名：输入用于登录代理服务器的用户名。
    • 密码：输入用于登录代理服务器的密码。

  3. 单击“确定”。

通过通知程序菜单设置网络代理首选项

在 Mac OS X 上不支持该方法。

  1. 在桌面通知区域的通知程序图标上单击鼠标右键，然后单击“首选项”。

    屏幕将显示“首选项”窗口。

  2. 按照上述说明继续操作。
