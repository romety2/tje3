设置通知程序首选项

通过 Update Tool 的“首选项”功能，您可以设置影响通知程序是否检查更新以及更新检查频率的首选项：

通过 Update Tool 设置通知程序首选项

  1. 在工具栏上，单击“首选项”。

    Update Tool 将显示“首选项”窗口。

  2. 单击“更新”选项卡。

  3. 选择通知程序自动检查已安装组件有无更新的频率。

  4. 单击“确定”。

通过通知程序菜单设置通知程序首选项

在 Mac OS X 上不支持该方法。

  1. 在桌面通知区域的通知程序图标上单击鼠标右键，然后单击“首选项”。

    屏幕将显示“首选项”窗口。

  2. 单击“更新”选项卡。

  3. 选择通知程序自动检查已安装组件有无更新的频率。

  4. 单击“确定”。

有关设置 Update Tool 和通知程序首选项的更多信息，请参见首选项概述部分。
