组件列表

组件列表显示在 Update Tool 窗口的右侧，工具栏的正下方。根据您查看的是安装映像的可用附加软件、可用更新还是已安装的组件，组件列表会显示不同的信息。

  • 可用附加软件视图

  • 可用更新视图

  • 已安装的组件视图

此外，如果在组件列表上单击鼠标右键（在 Macintosh 上是按住 Control 键并单击），会显示一个上下文菜单，该菜单中提供了对整个组件列表和您在该组件列表中选择的项目执行操作的命令。

类别

发布日期

附加软件的发布日期。

版本

附加软件的版本。

安装大小

附加软件的大约安装大小。

来源

可用更新视图

列

说明

组件

具有可用更新的组件的名称。

!

警报列，如果更新包含已安装组件的安全修复，将显示星号 (*)。

发布日期

更新的发布日期。

新版本

更新组件的版本。

已安装的版本

当前已安装组件的版本。

下载

更新的大约安装大小。请注意，由于更新中只包含修改的文件，因此，更新大小通常小于整个组件的安装大小。

如果不修改任何文件，只是修改其他属性（如文件权限、组件说明以及其他元数据），则“更改”值将为零。

如果更新会导致文件被删除或者减少一个或多个文件的大小，则“更改”值将为负数。

来源

将从中获取更新的系统信息库的名称。
